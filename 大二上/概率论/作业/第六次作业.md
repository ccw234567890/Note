## 🎯 问题 1 (Section 4.4, Problem 3)

### 题目
> 假设 X 是一个随机变量，其 $E(X)=1$， $E(X^{2})=2$， $E(X^{3})=5$。求 X 的第三中心矩的值。

### 知识点
>[!info] 核心概念
> **1. 矩 (Moment):** $k$ 阶原点矩定义为 $E(X^k)$。
> **2. 中心矩 (Central Moment):** $k$ 阶中心矩定义为 $\mu_k = E[(X - \mu)^k]$，其中 $\mu = E(X)$。
> **3. 第三中心矩:** $\mu_3 = E[(X - \mu)^3]$，用于衡量概率分布的**偏度 (Skewness)**。
> **4. 期望的线性性质:** $E[aX + bY + c] = aE[X] + bE[Y] + c$。

### 解析过程
>[!tip] 解题步骤
> 1. 写出第三中心矩的定义式 $\mu_3 = E[(X - \mu)^3]$。
> 2. 将 $\mu = E(X) = 1$ 代入，并使用二项式定理展开立方项。
> 3. 利用期望的线性性质，将展开后的期望拆开。
> 4. 代入已知值 $E(X)$, $E(X^2)$, $E(X^3)$ 并计算。

**详细计算:**

1.  根据定义，第三中心矩为：
    $$\mu_3 = E[(X - \mu)^3] \quad \text{其中 } \mu = E(X)$$

2.  已知 $\mu = E(X) = 1$。代入并展开：
    $$\mu_3 = E[(X - 1)^3] = E[X^3 - 3X^2 + 3X - 1]$$

3.  利用期望的线性性质：
    $$\mu_3 = E[X^3] - 3E[X^2] + 3E[X] - E[1]$$

4.  代入题目给定的值：$E(X)=1$, $E(X^2)=2$, $E(X^3)=5$。
    $$\mu_3 = 5 - 3(2) + 3(1) - 1$$
    $$\mu_3 = 5 - 6 + 3 - 1$$
    $$\mu_3 = 1$$

### 答案
>[!summary] 最终答案
> X 的第三中心矩的值为 **1**。

## 🎯 问题 2 (Section 4.4, Problem 6)

### 题目
> 假设 X 在区间 [a, b] 上服从均匀分布。求 X 的矩母函数 (m.g.f.)。

### 知识点
>[!info] 核心概念
> **1. 均匀分布 (Uniform Distribution):** 若 X 在 $[a, b]$ 上服从均匀分布，其 p.d.f. 为：
> $$f(x) = \begin{cases} \frac{1}{b-a} & \text{for } a \le x \le b \\ 0 & \text{otherwise} \end{cases}$$
> **2. 矩母函数 (m.g.f.):** 定义为 $M_X(t) = E[e^{tX}]$。
> **3. 连续型变量的期望:** $E[g(X)] = \int_{-\infty}^{\infty} g(x)f(x)dx$。

### 解析过程
>[!tip] 解题步骤
> 1. 写出 m.g.f. 的定义式 $M_X(t) = E[e^{tX}]$。
> 2. 根据连续型期望公式，写成积分形式 $M_X(t) = \int_{-\infty}^{\infty} e^{tx} f(x) dx$。
> 3. 代入均匀分布的 p.d.f. $f(x) = \frac{1}{b-a}$，积分区间为 $[a, b]$。
> 4. 计算定积分，注意讨论 $t=0$ 的情况。

**详细计算:**

1.  根据 m.g.f. 的定义：
    $$M_X(t) = E[e^{tX}]$$

2.  将其写成积分形式：
    $$M_X(t) = \int_{a}^{b} e^{tx} \cdot \left(\frac{1}{b-a}\right) dx$$

3.  计算积分：
    $$M_X(t) = \frac{1}{b-a} \int_{a}^{b} e^{tx} dx$$
    
    >[!note] 注意分情况讨论
    > 当 $t=0$ 时，分母为零，需要单独处理。
    
    * **当 $t \neq 0$ 时：**
        $$M_X(t) = \frac{1}{b-a} \left[ \frac{e^{tx}}{t} \right]_{x=a}^{x=b}$$
        $$M_X(t) = \frac{1}{b-a} \left( \frac{e^{tb} - e^{ta}}{t} \right) = \frac{e^{bt} - e^{at}}{t(b-a)}$$
    * **当 $t = 0$ 时：**
        $$M_X(0) = E[e^{0 \cdot X}] = E[1] = 1$$

### 答案
>[!summary] 最终答案
> X 的矩母函数 (m.g.f.) 为：
> $$M_X(t) = \begin{cases} \frac{e^{bt} - e^{at}}{t(b-a)} & \text{for } t \neq 0 \\ 1 & \text{for } t = 0 \end{cases}$$

## 🎯 问题 3 (Section 4.6, Problem 3)

### 题目
> 假设 X 在区间 [-2, 2] 上服从均匀分布，且 $Y=X^{6}$。证明 X 和 Y 不相关。

### 知识点
>[!info] 核心概念
> **1. 不相关 (Uncorrelated):** X 和 Y 不相关，当且仅当 $\text{Cov}(X, Y) = 0$。
> **2. 协方差 (Covariance):** $\text{Cov}(X, Y) = E[XY] - E[X]E[Y]$。
> **3. 奇函数与对称区间:** 若 $g(x)$ 是奇函数 (即 $g(-x) = -g(x)$)，则在对称区间 $[-c, c]$ 上的积分为 $\int_{-c}^{c} g(x) dx = 0$。

### 解析过程
>[!tip] 解题步骤
> 1. 目标是证明 $\text{Cov}(X, Y) = 0$。
> 2. 写出协方差公式 $\text{Cov}(X, Y) = E[XY] - E[X]E[Y]$。
> 3. 代入 $Y=X^6$，得到 $\text{Cov}(X, X^6) = E[X^7] - E[X]E[X^6]$。
> 4. 计算 $E[X]$ 和 $E[X^7]$。
> 5. 利用奇函数在对称区间上的积分特性。

**详细计算:**

1.  X 在 [-2, 2] 上的 p.d.f. 为：
    $$f(x) = \frac{1}{2 - (-2)} = \frac{1}{4}, \quad \text{for } -2 \le x \le 2$$

2.  计算 $E[X]$：
    $$E[X] = \int_{-2}^{2} x \cdot \frac{1}{4} dx$$
    >[!warning] 奇函数积分
    > 被积函数 $g(x) = \frac{x}{4}$ 是奇函数，积分区间 $[-2, 2]$ 对称，故积分为 0。
    $E[X] = 0$。

3.  计算 $E[XY] = E[X^7]$：
    $$E[X^7] = \int_{-2}^{2} x^7 \cdot \frac{1}{4} dx$$
    被积函数 $h(x) = \frac{x^7}{4}$ 也是奇函数，积分区间对称。
    因此，$E[X^7] = 0$。

4.  计算协方差：
    $$\text{Cov}(X, Y) = E[X^7] - E[X]E[X^6]$$
    $$\text{Cov}(X, Y) = 0 - (0) \cdot E[X^6] = 0$$

5.  因为 $\text{Cov}(X, Y) = 0$，所以 X 和 Y 不相关。

### 答案
>[!summary] 最终答案
> 1.  计算 $E[X] = \int_{-2}^{2} \frac{x}{4} dx = 0$（奇函数在对称区间上的积分为0）。
> 2.  计算 $E[XY] = E[X^7] = \int_{-2}^{2} \frac{x^7}{4} dx = 0$（同理）。
> 3.  计算协方差 $\text{Cov}(X, Y) = E[XY] - E[X]E[Y] = 0 - (0) \cdot E[Y] = 0$。
> 4.  由于协方差为 0，**因此 X 和 Y 是不相关的**。

## 🎯 问题 4 (Section 4.6, Problem 12)

### 题目
> 假设 X 和 Y 有一个连续的联合分布，其联合 p.d.f. 如下：
> $f(x,y)=\begin{cases}\frac{1}{3}(x+y)&for~0\le x\le1~and~0\le y\le2,\\ 0&otherwise.\end{cases}$
> 求 $\text{Var}(2X - 3Y + 8)$ 的值。

### 知识点
>[!info] 核心概念
> **1. 方差的性质:**
> $$\text{Var}(aX + bY + c) = a^2 \text{Var}(X) + b^2 \text{Var}(Y) + 2ab \cdot \text{Cov}(X, Y)$$
> **2. 方差与协方差:**
> * $\text{Var}(X) = E[X^2] - (E[X])^2$
> * $\text{Var}(Y) = E[Y^2] - (E[Y])^2$
> * $\text{Cov}(X, Y) = E[XY] - E[X]E[Y]$
> **3. 边缘 P.D.F. (Marginal p.d.f.):**
> * $f_X(x) = \int_{-\infty}^{\infty} f(x,y) dy$
> * $f_Y(y) = \int_{-\infty}^{\infty} f(x,y) dx$

### 解析过程
>[!tip] 解题步骤
> 1.  目标是计算 $4\text{Var}(X) + 9\text{Var}(Y) - 12\text{Cov}(X, Y)$。
> 2.  计算 $E[X]$, $E[X^2]$ 进而得到 $\text{Var}(X)$。
> 3.  计算 $E[Y]$, $E[Y^2]$ 进而得到 $\text{Var}(Y)$。
> 4.  计算 $E[XY]$ 进而得到 $\text{Cov}(X, Y)$。
> 5.  汇总计算。

**详细计算:**

1.  **求 $E[X]$ 和 $E[X^2]$:**
    * $f_X(x) = \int_{0}^{2} \frac{1}{3}(x+y) dy = \frac{1}{3} [xy + \frac{y^2}{2}]_0^2 = \frac{1}{3}(2x + 2) = \frac{2}{3}(x+1)$, ($0 \le x \le 1$)
    * $E[X] = \int_{0}^{1} x \cdot \frac{2}{3}(x+1) dx = \frac{2}{3} \int_{0}^{1} (x^2+x) dx = \frac{2}{3} [\frac{x^3}{3} + \frac{x^2}{2}]_0^1 = \frac{2}{3}(\frac{1}{3} + \frac{1}{2}) = \frac{5}{9}$
    * $E[X^2] = \int_{0}^{1} x^2 \cdot \frac{2}{3}(x+1) dx = \frac{2}{3} \int_{0}^{1} (x^3+x^2) dx = \frac{2}{3} [\frac{x^4}{4} + \frac{x^3}{3}]_0^1 = \frac{2}{3}(\frac{1}{4} + \frac{1}{3}) = \frac{7}{18}$
    * $\text{Var}(X) = E[X^2] - (E[X])^2 = \frac{7}{18} - (\frac{5}{9})^2 = \frac{7}{18} - \frac{25}{81} = \frac{63 - 50}{162} = \frac{13}{162}$

2.  **求 $E[Y]$ 和 $E[Y^2]$:**
    * $f_Y(y) = \int_{0}^{1} \frac{1}{3}(x+y) dx = \frac{1}{3} [\frac{x^2}{2} + xy]_0^1 = \frac{1}{3}(\frac{1}{2} + y)$, ($0 \le y \le 2$)
    * $E[Y] = \int_{0}^{2} y \cdot \frac{1}{3}(\frac{1}{2} + y) dy = \frac{1}{3} \int_{0}^{2} (\frac{y}{2} + y^2) dy = \frac{1}{3} [\frac{y^2}{4} + \frac{y^3}{3}]_0^2 = \frac{1}{3}(1 + \frac{8}{3}) = \frac{11}{9}$
    * $E[Y^2] = \int_{0}^{2} y^2 \cdot \frac{1}{3}(\frac{1}{2} + y) dy = \frac{1}{3} \int_{0}^{2} (\frac{y^2}{2} + y^3) dy = \frac{1}{3} [\frac{y^3}{6} + \frac{y^4}{4}]_0^2 = \frac{1}{3}(\frac{8}{6} + \frac{16}{4}) = \frac{16}{9}$
    * $\text{Var}(Y) = E[Y^2] - (E[Y])^2 = \frac{16}{9} - (\frac{11}{9})^2 = \frac{144 - 121}{81} = \frac{23}{81}$

3.  **求 $\text{Cov}(X, Y)$:**
    * $E[XY] = \int_{0}^{1} \int_{0}^{2} xy \cdot \frac{1}{3}(x+y) dy dx = \frac{1}{3} \int_{0}^{1} \int_{0}^{2} (x^2y + xy^2) dy dx$
    * $= \frac{1}{3} \int_{0}^{1} [x^2\frac{y^2}{2} + x\frac{y^3}{3}]_0^2 dx = \frac{1}{3} \int_{0}^{1} (2x^2 + \frac{8}{3}x) dx = \frac{1}{3} [\frac{2x^3}{3} + \frac{4x^2}{3}]_0^1 = \frac{1}{3}(\frac{2}{3} + \frac{4}{3}) = \frac{2}{3}$
    * $\text{Cov}(X, Y) = E[XY] - E[X]E[Y] = \frac{2}{3} - (\frac{5}{9})(\frac{11}{9}) = \frac{54 - 55}{81} = -\frac{1}{81}$

4.  **汇总:**
    * $\text{Var}(2X - 3Y + 8) = 4\text{Var}(X) + 9\text{Var}(Y) - 12\text{Cov}(X, Y)$
    * $= 4 (\frac{13}{162}) + 9 (\frac{23}{81}) - 12 (-\frac{1}{81})$
    * $= \frac{26}{81} + \frac{207}{81} + \frac{12}{81} = \frac{26 + 207 + 12}{81} = \frac{245}{81}$

### 答案
>[!summary] 最终答案
> $\text{Var}(2X - 3Y + 8)$ 的值为 **$\frac{245}{81}$**。

## 🎯 问题 5 (Section 4.6, Problem 13)

### 题目
> 假设 X 和 Y 是随机变量，使得 $\text{Var}(X)=9$, $\text{Var}(Y)=4$, 且 $\rho(X,Y)=-1/6$。
> 求 (a) $\text{Var}(X+Y)$ 和 (b) $\text{Var}(X-3Y+4)$。

### 知识点
>[!info] 核心概念
> **1. 相关系数 (Correlation Coefficient):** $\rho(X,Y) = \frac{\text{Cov}(X,Y)}{\sqrt{\text{Var}(X)\text{Var}(Y)}}$
>    因此, $\text{Cov}(X,Y) = \rho(X,Y) \cdot \sqrt{\text{Var}(X)\text{Var}(Y)}$。
> **2. 方差的线性组合:**
> * $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$
> * $\text{Var}(aX + bY + c) = a^2 \text{Var}(X) + b^2 \text{Var}(Y) + 2ab \cdot \text{Cov}(X, Y)$

>[!warning] 重要提示
> 注意：常数 $c$ 不影响方差，即 $\text{Var}(Z+c) = \text{Var}(Z)$。

### 解析过程
>[!tip] 解题步骤
> 1.  **求 $\text{Cov}(X, Y)$:**
>     * $\text{Cov}(X,Y) = \rho(X,Y) \cdot \sqrt{\text{Var}(X)\text{Var}(Y)}$
>     * $\text{Cov}(X,Y) = (-\frac{1}{6}) \cdot \sqrt{9 \cdot 4} = (-\frac{1}{6}) \cdot 6 = -1$。
> 2.  **求解 (a) $\text{Var}(X+Y)$:**
>     * $\text{Var}(X+Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X, Y)$
>     * $\text{Var}(X+Y) = 9 + 4 + 2(-1) = 13 - 2 = 11$。
> 3.  **求解 (b) $\text{Var}(X-3Y+4)$:**
>     * $\text{Var}(X-3Y+4) = \text{Var}(1 \cdot X + (-3) \cdot Y + 4)$
>     * $= (1)^2 \text{Var}(X) + (-3)^2 \text{Var}(Y) + 2(1)(-3) \text{Cov}(X, Y)$
>     * $= 1 \cdot \text{Var}(X) + 9 \cdot \text{Var}(Y) - 6 \cdot \text{Cov}(X, Y)$
>     * $= 1(9) + 9(4) - 6(-1) = 9 + 36 + 6 = 51$。

### 答案
>[!summary] 最终答案
> (a) $\text{Var}(X+Y)$ =  **11**
> (b) $\text{Var}(X-3Y+4)$ =  **51**

## 🎯 问题 6 (Section 4.6, Problem 18)

### 题目
> 假设 X 和 Y 有一个连续分布，其联合 p.d.f. 如下：
> $f(x,y)=\begin{cases}x+y&for~0\le x\le1~and~0\le y\le1,\\ 0&otherwise.\end{cases}$
> 计算协方差 $\text{Cov}(X,Y)$。

### 知识点
>[!info] 核心概念
> **1. 协方差 (Covariance):** $\text{Cov}(X, Y) = E[XY] - E[X]E[Y]$。
> **2. 边缘 P.D.F. (Marginal p.d.f.):**
> * $f_X(x) = \int_{0}^{1} f(x,y) dy$
> * $f_Y(y) = \int_{0}^{1} f(x,y) dx$
> **3. 期望计算:**
> * $E[X] = \int_{0}^{1} x f_X(x) dx$
> * $E[Y] = \int_{0}^{1} y f_Y(y) dy$
> * $E[XY] = \int_{0}^{1} \int_{0}^{1} xy f(x,y) dy dx$

### 解析过程
>[!tip] 解题步骤
> 1.  目标是计算 $\text{Cov}(X, Y) = E[XY] - E[X]E[Y]$。
> 2.  计算 $E[X]$ (通过先求边缘 $f_X(x)$)。
> 3.  利用对称性求 $E[Y]$。
> 4.  计算 $E[XY]$。
> 5.  汇总计算。

**详细计算:**

1.  **计算 $E[X]$:**
    * $f_X(x) = \int_{0}^{1} (x+y) dy = [xy + \frac{y^2}{2}]_0^1 = x + \frac{1}{2}$, ($0 \le x \le 1$)
    * $E[X] = \int_{0}^{1} x(x + \frac{1}{2}) dx = \int_{0}^{1} (x^2 + \frac{x}{2}) dx = [\frac{x^3}{3} + \frac{x^2}{4}]_0^1 = \frac{1}{3} + \frac{1}{4} = \frac{7}{12}$。

2.  **计算 $E[Y]$:**
    >[!note] 利用对称性
    > 由于 $f(x,y)$ 和积分区域在 $x$ 和 $y$ 上是对称的 (即 $f(x,y) = f(y,x)$ 且 $x, y$ 范围相同)，所以 $X$ 和 $Y$ 的边缘分布和期望是相同的。
    因此，$E[Y] = E[X] = \frac{7}{12}$。

3.  **计算 $E[XY]$:**
    * $E[XY] = \int_{0}^{1} \int_{0}^{1} xy(x+y) dy dx = \int_{0}^{1} \int_{0}^{1} (x^2y + xy^2) dy dx$
    * $= \int_{0}^{1} [x^2\frac{y^2}{2} + x\frac{y^3}{3}]_0^1 dx = \int_{0}^{1} (\frac{x^2}{2} + \frac{x}{3}) dx$
    * $= [\frac{x^3}{6} + \frac{x^2}{6}]_0^1 = \frac{1}{6} + \frac{1}{6} = \frac{2}{6} = \frac{1}{3}$。

4.  **计算 $\text{Cov}(X, Y)$:**
    * $\text{Cov}(X, Y) = E[XY] - E[X]E[Y]$
    * $= \frac{1}{3} - (\frac{7}{12})(\frac{7}{12}) = \frac{1}{3} - \frac{49}{144}$
    * $= \frac{48}{144} - \frac{49}{144} = -\frac{1}{144}$。

### 答案
>[!summary] 最终答案
> 协方差 $\text{Cov}(X,Y)$ 的值为 **$-\frac{1}{144}$**。
