# 物理笔记：量子论的诞生 (Quantum Theory)

> [!abstract] **课程概要**
> 本章探讨了经典物理在解释微观现象时的失效，介绍了普朗克量子假说、爱因斯坦光子理论、康普顿效应、波粒二象性以及原子模型的演变。
> **核心思想**：能量不是连续的，而是离散的（量子化）。

---

## 1. 热辐射 (Thermal Radiation)

### 1.1 基本概念
* **现象**：任何温度 $T > 0 K$（绝对零度）的物体都会发射电磁波。
* **微观机制**：温度代表粒子的平均动能。带电粒子（如电子）的热运动（振动/加速）会产生变化的电磁场，从而向外辐射电磁波。
* **例子**：
    * **红外热像仪**：利用物体辐射的红外线成像。
    * **烧红的铁**：低温时是黑色 $\to$ 红 $\to$ 橙 $\to$ 黄 $\to$ 白（白炽）。颜色代表了温度。

### 1.2 黑体 (Blackbody)
为了研究纯粹的辐射规律，排除反射光的干扰，物理学家定义了**黑体**。
* **定义**：一个理想模型，它**吸收所有**照射到它表面的电磁波（不反射），同时自身进行完美的辐射。
* **实验模型（空腔辐射）**：
    * 在一个球形空腔上开一个小孔。
    * 光射入小孔后在内部多次反射被完全吸收。
    * 从小孔出来的光就是最纯粹的**黑体辐射**。

### 1.3 辐射定律
通过测量黑体辐射光谱（强度 vs 波长），发现了以下规律：

#### A. 维恩位移定律 (Wien's Displacement Law)
* **规律**：温度 $T$ 越高，辐射最强处的波长 $\lambda_{max}$ 越短。
* **公式**：
    $$\lambda_{max} \propto \frac{1}{T}$$
* **应用实例**：
    1.  **测量太阳温度**：我们在地面测得太阳辐射峰值波长，代入公式算出太阳表面温度约为 **6000 K**。
    2.  **灯泡的色温 (Color Temperature)**：
        * **6000 K（冷光）**：像正午阳光，含蓝紫光多（波长短，频率高，能量高）。
        * **3000 K（暖光）**：像烛光，含红橙光多（波长长，频率低）。
        * *注意*：生活中感觉“暖”的光，其实物理温度（色温）较低；感觉“冷”的光，色温反而高。

#### B. 斯特藩-玻尔兹曼定律 (Stefan-Boltzmann Law)
* **规律**：单位面积、单位时间辐射的总能量（功率密度）与温度的四次方成正比。
* **公式**：
    $$I = \sigma T^4$$
    * $\sigma$：斯特藩常数。
* **课堂练习：计算人体辐射功率**
    * 人体的表面积 $S$ 约为 $1.5 - 2 m^2$。
    * 体温 $T \approx 310 K$。
    * 总功率 $P = S \cdot \sigma T^4$。
    * *结果*：约为几百瓦（虽然我们也在吸收环境辐射，但在红外波段我们是个很强的辐射源）。

### 1.4 紫外灾难与普朗克公式
* **经典理论失败（紫外灾难）**：瑞利-金斯公式（基于经典电磁学和热力学）在长波段符合实验，但在**短波段（高频/紫外区）**预测辐射强度趋于无穷大。
* **普朗克假说 (1900年)**：
    * 为了凑出符合实验的公式，普朗克假设：**能量不是连续的，而是一份一份的**。
    * 能量量子化公式：
        $$E = n h f \quad (n=1, 2, 3...)$$
        * $h$：普朗克常数 ($6.626 \times 10^{-34} J\cdot s$)。
        * $f$：频率。
    * **结果**：普朗克公式完美拟合了全波段的实验曲线。

## 2. 光电效应 (Photoelectric Effect)

### 2.1 实验现象
用光照射金属表面，电子从金属中逸出（光电子）。
* **关键实验量**：
    * **截止电压 ($V_s$)**：加反向电压，当电流刚好为0时，$e V_s = K_{max}$（最大初动能）。
    * **饱和电流**：正向电压增加到一定程度，电流不再增加。

### 2.2 经典理论 vs 实验结果

| 比较项目 | 经典波动理论预测 | 实验实际结果 |
| :--- | :--- | :--- |
| **动能与光强** | 光越强（振幅大），电子动能越大。 | $K_{max}$ **只与频率有关**，与光强无关。 |
| **发射条件** | 只要光够强，任何频率都能打出电子。 | 存在**截止频率 ($f_c$)**，低于此频率，光再强也没用。 |
| **发射时间** | 需要时间积累能量（受迫振动）。 | 电子**瞬间逸出** ($<10^{-9}s$)。 |

### 2.3 爱因斯坦的光子理论
爱因斯坦提出光就是粒子流（光子）。
* **光子能量**：$E = hf$。
* **光电效应方程**：
    $$hf = W_0 + K_{max}$$
    * $hf$：入射光子能量（一份能量全给一个电子）。
    * $W_0$：**逸出功**（电子挣脱原子核束缚所需最小能量）。
    * $K_{max}$：电子出来的最大动能。
* **推导关系**：
    $$e V_s = hf - W_0$$
    * 这解释了为什么截止电压 $V_s$ 与频率 $f$ 成线性关系，斜率为 $h/e$。

> [!example] **通俗解释：为什么会有饱和电流？**
> * 光强代表单位时间射入的**光子数量**。
> * 如果频率不变，光强增加 $\to$ 光子数增加 $\to$ 被打出的电子数增加 $\to$ 电流增加。
> * 但在固定光强下，每秒光子数是固定的，产生的电子数也是上限的。当电压足够大，把所有出来的电子都拉过去了，电流就不能再大了（饱和）。

> [!tip] **能量量级与人体伤害**
> * **可见光**：能量约为几 eV（电子伏特）。这对人体是安全的，因为能量不足以破坏分子键。
> * **高温物体（如开水）**：约 $70^\circ C$ 以上会烫伤，破坏细胞结构。
> * **紫外线/X射线**：单光子能量很大，超过了分子化学键能。
>     * 例子：西藏高原紫外线强，人容易皮肤黝黑甚至灼伤（像被开水烫过）。
>     * 蓝光（频率高）比红光更容易伤害眼睛。

## 3. 康普顿效应 (Compton Effect)

### 3.1 实验描述
使用 **X射线** 轰击石墨（含有自由电子）。
* **现象**：散射光中除了有原波长 $\lambda_0$ 的光，还出现了**波长变长**的光 $\lambda'$。
* **规律**：波长改变量 $\Delta \lambda$ 随散射角 $\phi$ 增大而增大。

### 3.2 经典 vs 量子解释
* **经典失败**：经典理论认为电子做受迫振动，发射频率应等于入射频率，波长不该变。
* **量子解释**：
    * **粒子碰撞模型**：光子（粒子）与**自由静止电子**发生**弹性碰撞**。
    * 光子把部分能量传给电子 $\to$ 光子能量变小 $\to$ 频率变低 $\to$ **波长变长**。

### 3.3 详细推导（必考）
> [!NOTE] **模型假设**
> 1.  光子：$E=hf$, $p=h/\lambda$。
> 2.  电子：碰撞前静止（自由电子），碰撞后反冲。
> 3.  使用相对论能量动量关系。

**Step 1: 能量守恒**
$$hf + m_0c^2 = hf' + E_e$$
其中 $E_e = \sqrt{(p_e c)^2 + (m_0 c^2)^2}$ 是反冲电子的总能量。
整理得：
$$E_e = h(f - f') + m_0c^2 \quad \text{......(1)}$$

**Step 2: 动量守恒（矢量三角形）**
利用余弦定理：
$$p_e^2 = p_0^2 + p'^2 - 2p_0 p' \cos\phi$$
两边乘 $c^2$（因为 $p=hf/c$）：
$$(p_e c)^2 = (hf)^2 + (hf')^2 - 2(hf)(hf') \cos\phi \quad \text{......(2)}$$

**Step 3: 联立求解**
由相对论公式：$E_e^2 = (p_e c)^2 + (m_0 c^2)^2$。
将(1)式平方，代入：
$$[h(f-f') + m_0c^2]^2 = (p_e c)^2 + (m_0 c^2)^2$$
展开左边：
$$h^2(f-f')^2 + 2h(f-f')m_0c^2 + (m_0c^2)^2 = (p_e c)^2 + (m_0c^2)^2$$
消去 $(m_0c^2)^2$，代入(2)式的 $(p_e c)^2$：
$$h^2(f^2 - 2ff' + f'^2) + 2h(f-f')m_0c^2 = h^2f^2 + h^2f'^2 - 2h^2ff'\cos\phi$$
消去 $h^2f^2$ 和 $h^2f'^2$：
$$-2h^2ff' + 2h(f-f')m_0c^2 = -2h^2ff'\cos\phi$$
整理：
$$m_0c^2(f-f') = hff'(1-\cos\phi)$$
两边除以 $ff'm_0c$：
$$\frac{c}{f'} - \frac{c}{f} = \frac{h}{m_0 c}(1-\cos\phi)$$
因为 $\lambda = c/f$，得到最终公式：

> [!important] **康普顿散射公式**
> $$\Delta \lambda = \lambda' - \lambda_0 = \frac{h}{m_0 c}(1-\cos\phi)$$
> 利用半角公式 $1-\cos\phi = 2\sin^2(\frac{\phi}{2})$，也可以写成：
> $$\Delta \lambda = \frac{2h}{m_0 c} \sin^2\left(\frac{\phi}{2}\right)$$
> * $\lambda_c = \frac{h}{m_0 c} \approx 0.00243 nm$ 叫**电子康普顿波长**。

### 3.4 两个关键问题
1.  **为什么可见光看不到康普顿效应？**
    * $\lambda_c$ 只有 $10^{-12}m$ 量级。
    * 可见光波长 $\sim 10^{-7}m$，变化量 $\Delta \lambda$ 相比原波长太小，就像身家一千万丢了5毛钱，根本察觉不到。
    * 必须用波长极短的 **X射线** 才能观测到。
2.  **为什么实验中还有波长不变的射线？**
    * 光子不仅仅撞击自由电子，还会撞击**内层电子（束缚电子）**。
    * 内层电子与原子核紧密结合，光子相当于撞击**整个原子**。
    * 原子的质量 $M \gg m_{electron}$。公式中的 $m_0$ 换成 $M$，$\Delta \lambda \to 0$。
    * 就像乒乓球撞在墙上，能量几乎不损失，波长不变。

## 4. 波粒二象性 (Wave-Particle Duality)

* **光**：既有波动性（干涉、衍射），又有粒子性（光电效应、康普顿效应）。
* **物质（德布罗意波）**：
    * 1924年，德布罗意提出大胆猜想：实物粒子（如电子）也有波动性。
    * **公式**：
        $$\lambda = \frac{h}{p} = \frac{h}{mv}$$
    * **验证**：戴维森-革末实验（电子束衍射）证明了电子确实有波长。
    * **宏观物体**：质量 $m$ 太大 $\to$ 波长 $\lambda$ 极小 $\to$ 波动性不可见。

## 5. 原子结构模型 (Structure of the Atom)

### 5.1 汤姆孙模型（枣糕模型）
* 原子是一个正电荷球体，电子像葡萄干一样镶嵌其中。
* **预测**：$\alpha$ 粒子轰击金箔应直接穿过，不发生大偏转。

### 5.2 卢瑟福模型（核式结构）
* **实验**：$\alpha$ 粒子散射实验。
* **现象**：
    1.  绝大多数直接穿过（说明原子内部很空）。
    2.  极少数发生大角度偏转甚至反弹（说明中心有质量很大、体积很小的正电荷核心）。
* **结论**：原子核居中，电子绕核运动（像太阳系）。

### 5.3 卢瑟福模型的致命缺陷
* **经典电磁理论**：电子做圆周运动有加速度 $\to$ 必须向外辐射电磁波 $\to$ 能量损失。
* **后果**：电子轨道半径会越来越小，最终**螺旋落入原子核**。
* **矛盾**：现实中的原子非常稳定。
* **解决**：这预示着需要新的理论（玻尔模型）来解释原子的稳定性。
