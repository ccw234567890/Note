## 1. 基础算符与量子化规则 (Operators)

在量子力学中，我们将经典力学的物理量（Physical Quantities）替换为对应的**算符（Operators）**，作用于波函数 $\Psi(x,t)$。

### 1.1 核心算符
| 物理量 | 符号 | 经典表达式 | 量子力学算符 |
| :--- | :---: | :---: | :--- |
| **能量 (Energy)** | $E$ | $E$ | $\hat{E} = i\hbar \frac{\partial}{\partial t}$ |
| **动量 (Momentum)** | $p$ | $p$ | $\hat{p} = -i\hbar \nabla$ (一维: $-i\hbar \frac{\partial}{\partial x}$) |
| **位置 (Position)** | $x$ | $x$ | $\hat{x} = x$ (仅乘以 $x$) |
| **势能 (Potential)** | $U$ | $U(x)$ | $\hat{U} = U(x)$ (仅乘以 $U(x)$) |

### 1.2 导出算符
基于上述基础算符，利用经典关系式导出：
* **动能算符 (Kinetic Energy Operator)**:
    经典关系: $E_k = \frac{p^2}{2m}$
    量子算符:
    $$\hat{T} = \frac{\hat{p}^2}{2m} = \frac{(-i\hbar \frac{\partial}{\partial x})^2}{2m} = -\frac{\hbar^2}{2m} \frac{\partial^2}{\partial x^2}$$
    *(三维情况为 $-\frac{\hbar^2}{2m} \nabla^2$, 其中 $\nabla^2$ 为拉普拉斯算子)*

* **角动量算符 (Angular Momentum)**:
    经典关系: $\vec{L} = \vec{r} \times \vec{p}$
    量子算符: $\hat{L} = -i\hbar (\vec{r} \times \nabla)$

* **哈密顿算符 (Hamiltonian Operator)**:
    代表系统的**总能量**。
    $$\hat{H} = \hat{T} + \hat{U} = -\frac{\hbar^2}{2m} \nabla^2 + U(\vec{r})$$

## 2. 薛定谔方程 (Schrödinger Equation)

### 2.1 含时薛定谔方程 (General Form)
通过能量算符作用于波函数得到：
$$i\hbar \frac{\partial \Psi}{\partial t} = \hat{H} \Psi$$
展开即为：
$$i\hbar \frac{\partial \Psi(\vec{r},t)}{\partial t} = \left[ -\frac{\hbar^2}{2m} \nabla^2 + U(\vec{r},t) \right] \Psi(\vec{r},t)$$

### 2.2 分离变量法与定态薛定谔方程
**假设前提**: 势能 $U$ 不随时间变化，即 $U = U(\vec{r})$ (保守力场)。

**步骤**:
1.  假设波函数可以分离为空间部分和时间部分的乘积：
    $$\Psi(\vec{r},t) = \psi(\vec{r}) f(t)$$
2.  代入方程，两边同除以 $\psi f$：
    $$i\hbar \frac{1}{f(t)} \frac{df(t)}{dt} = \frac{1}{\psi(\vec{r})} \hat{H} \psi(\vec{r})$$
3.  **分离变量**: 左边只含 $t$，右边只含 $\vec{r}$。要使两边恒等，它们必须等于同一个常数，这个常数就是**总能量 $E$**。

**得到两个方程**:
> [!NOTE] 1. 时间部分方程
> $$i\hbar \frac{df}{dt} = E f \implies f(t) = C e^{-iEt/\hbar}$$
> *这表明随时间的变化只是相位的变化，模长不变。*

> [!abstract] 2. 定态薛定谔方程 (Time-Independent SE)
> $$\hat{H} \psi(\vec{r}) = E \psi(\vec{r})$$
> 展开一维形式:
> $$-\frac{\hbar^2}{2m} \frac{d^2 \psi(x)}{dx^2} + U(x)\psi(x) = E\psi(x)$$

### 2.3 定态 (Stationary State) 的物理意义
* 解出的 $\psi_n$ 称为**本征函数 (Eigenfunction)**，对应的 $E_n$ 称为**本征能量 (Eigenenergy)**。
* 称为“定态”是因为其概率密度 $|\Psi|^2$ 不随时间变化：
    $$|\Psi|^2 = \Psi^* \Psi = (\psi^* e^{+iEt/\hbar}) (\psi e^{-iEt/\hbar}) = \psi^*(x)\psi(x) = |\psi(x)|^2$$
    *物理意义：在定态下，粒子在空间各点出现的概率分布是**静止**的。*

### 2.4 态叠加原理 (Superposition Principle)
薛定谔方程是**线性方程**。
如果 $\psi_1$ 和 $\psi_2$ 都是方程的解，那么它们的线性组合也是解：
$$\Psi = c_1 \psi_1 + c_2 \psi_2$$
此时系统处于“混合态”，粒子既处于状态 1 也处于状态 2。
* $|c_1|^2$：粒子处于状态 1 的概率。
* $|c_2|^2$：粒子处于状态 2 的概率。

## 3. 波函数的限制条件 (Standard Conditions)

为了使波函数具有物理意义（表示概率幅），它必须满足以下条件：
1.  **连续性 (Continuous)**: $\psi(x)$ 和其一阶导数 $\psi'(x)$ 必须处处连续（除非势能无穷大，导数可不连续）。
2.  **有限性 (Finite)**: 波函数不能发散（趋于无穷），否则无法归一化。
3.  **单值性 (Single-valued)**: 某一点的概率幅只能有一个值。
4.  **归一化 (Normalizable)**: 全空间的概率总和必须为 1。
    $$\int_{-\infty}^{+\infty} |\psi(x)|^2 dx = 1$$

## 4. 重点模型：一维无限深势井 (1D Infinite Potential Well)

> [!tip] 考试重点
> 这是一个最简单的模型，但包含了量子力学的核心概念：**能量量子化**。

### 4.1 模型设定
一个粒子被限制在宽度为 $L$ 的区域内运动（$0 \le x \le L$）。
**势能函数**:
$$
U(x) = 
\begin{cases} 
0, & 0 < x < L \quad (\text{井内，自由粒子}) \\
\infty, & x \le 0, x \ge L \quad (\text{井外，无法进入})
\end{cases}
$$

**物理推论**:
* 井外势能无穷大 $\to$ 粒子不可能出现在井外 $\to$ 井外 $\psi(x) = 0$。
* **边界条件 (Boundary Conditions)**: 由于波函数连续，在井壁处必须为 0。
    $$\psi(0) = 0, \quad \psi(L) = 0$$

### 4.2 求解定态薛定谔方程
在井内 ($0 < x < L$)，$U(x)=0$，方程变为：
$$-\frac{\hbar^2}{2m} \frac{d^2\psi}{dx^2} = E\psi \implies \frac{d^2\psi}{dx^2} + \frac{2mE}{\hbar^2}\psi = 0$$

令 $k^2 = \frac{2mE}{\hbar^2}$ ($E>0$，所以 $k$ 为实数)，方程化为简谐振动形式：
$$\psi'' + k^2\psi = 0$$

**通解**:
$$\psi(x) = A \sin(kx) + B \cos(kx)$$

### 4.3 代入边界条件定参
1.  **代入 $x=0$**:
    $$\psi(0) = A \sin(0) + B \cos(0) = B = 0$$
    $\therefore B=0$，波函数形式简化为 $\psi(x) = A \sin(kx)$。

2.  **代入 $x=L$**:
    $$\psi(L) = A \sin(kL) = 0$$
    * $A$ 不能为 0（否则波函数处处为 0，粒子不存在）。
    * $\therefore \sin(kL) = 0$。

**特征值求解**:
$$kL = n\pi \implies k_n = \frac{n\pi}{L}, \quad n = 1, 2, 3, ...$$
> [!WARNING] 关键点：为什么 $n \neq 0$？
> 如果 $n=0$，则 $k=0$，导致 $\psi(x)=0$（粒子消失），这在物理上无意义。因此 $n$ 必须取正整数。此处的 $n$ 称为**量子数 (Quantum Number)**。

### 4.4 结果分析

#### A. 能量量子化 (Energy Quantization)
将 $k_n$ 代回 $E$ 的表达式：
$$E_n = \frac{\hbar^2 k_n^2}{2m} = \frac{\hbar^2 (n\pi/L)^2}{2m}$$
整理得：
$$E_n = \frac{n^2 \pi^2 \hbar^2}{2mL^2}, \quad n=1,2,3...$$

* **结论**: 能量是不连续的（离散的）。
* **零点能 (Zero-point Energy)**: 
    当 $n=1$ 时，基态能量 $E_1 = \frac{\pi^2 \hbar^2}{2mL^2} \neq 0$。
    *这与经典力学不同，经典力学中粒子可以静止 ($E=0$)，但在量子力学中，由于测不准原理，被束缚的微观粒子永远无法静止。*

#### B. 波函数归一化
利用归一化条件求系数 $A$：
$$\int_0^L |\psi(x)|^2 dx = \int_0^L A^2 \sin^2(\frac{n\pi x}{L}) dx = 1$$
积分结果为 $A^2 \frac{L}{2} = 1 \implies A = \sqrt{\frac{2}{L}}$。

**最终波函数**:
$$\psi_n(x) = \sqrt{\frac{2}{L}} \sin\left(\frac{n\pi x}{L}\right)$$

#### C. 波函数图像与概率分布
* **$n=1$ (基态)**: 像半个正弦波，中间概率最大，没有节点。
* **$n=2$ (第一激发态)**: 一个完整的正弦波，中间 $x=L/2$ 处概率为 0（节点）。
* **$n=3$**: 1.5 个波长，有两个节点。
* *观察*: 概率分布是不均匀的，且呈现驻波形态。

## 5. 快速推导法：驻波模型 (Exam Hack)

> [!success] 老师推荐：考试必备技巧
> 如果考试记不住复杂的微分方程推导，可以用“德布罗意波+驻波条件”快速推导能量公式。

**逻辑链条**:
1.  **驻波条件**: 粒子被限制在井内来回反射，形成驻波。两端固定为波节。
    $$L = n \frac{\lambda}{2} \implies \lambda_n = \frac{2L}{n} \quad (n=1,2,3...)$$
2.  **德布罗意关系 (Wave-Particle Duality)**:
    $$p = \frac{h}{\lambda} = \frac{h}{2L/n} = \frac{nh}{2L}$$
3.  **能量关系**: 井内势能为 0，只有动能。
    $$E = \frac{p^2}{2m} = \frac{(nh/2L)^2}{2m} = \frac{n^2 h^2}{8mL^2}$$
    *(注: $\hbar = h/2\pi$，化简后与薛定谔方程解出的 $E_n = \frac{n^2 \pi^2 \hbar^2}{2mL^2}$ 完全一致)*

**这个方法的物理图像**:
粒子像波一样在井壁间反射，干涉形成驻波。因为能量不传播出井外，所以必须是驻波。

## 6. 验证海森堡测不准原理 (Uncertainty Principle Check)

对于无限深势井中的粒子：
* **位置不确定度**: 粒子在 $0$ 到 $L$ 之间，粗略估计 $\Delta x \sim L$。
* **动量不确定度**: 动量大小是确定的 $|p_n| = \hbar k_n$，但方向不确定（向左或向右反射）。
    $\Delta p \approx 2|p| = \frac{2h}{\lambda} = \frac{nh}{L}$ (此处仅做量级估计)。
    或者更严谨地，$\Delta p \sim \frac{\hbar}{L}$ (基态)。
    
**验证**:
$$\Delta x \cdot \Delta p \sim L \cdot \frac{\hbar}{L} \sim \hbar > \frac{\hbar}{2}$$
满足测不准原理。这也解释了为什么存在**零点能**：如果能量为 0，则动量 $p=0$ 且位置确定，这将违反测不准原理。
