### Page 3: 核心概念引入
![[Lecture 14.pdf#page=3]]
**讲解：**
- **什么是二阶电路？**
    - 电路里有两个储能元件（L和C，或者两个C，或者两个L）。
    - 描述这种电路需要用**二阶微分方程**。
- **两种响应类型**：
    1.  **零输入响应 (Zero Input Response)**：没有外接电源，只靠L和C里的初始能量“内耗”。能量耗尽，响应归零。
    2.  **有输入响应**：电路接入了电源。
        - **解的构成** = **齐次解 (Complementary)** + **特解 (Particular)**。
        - *齐次解*：对应暂态过程（自然响应），最终会消失。
        - *特解*：对应稳态过程（强迫响应），与输入信号形状相似。

---

### Page 4: 数学解法 (1) - 方程形式
![[Lecture 14.pdf#page=4]]
**讲解：**
- **标准方程**：
    $$\frac{d^2v}{dt^2} + a_1\frac{dv}{dt} + a_0v = b_0$$
    - 这是一个二阶常系数线性微分方程。
- **解的结构**：$v(t) = v_c(t) + v_p(t)$。
    - $v_c(t)$：**齐次解**（Complementary），令方程右边为0求得。取决于电路的“性格”（阻尼状态）。
    - $v_p(t)$：**特解**（Particular），取决于输入信号 $b_0$。

---

### Page 5: 数学解法 (2) - 求特解
![[Lecture 14.pdf#page=5]]
**讲解：**
- **特解怎么求？**
    - 如果输入是直流（常数），那么特解也是一个常数 $K$。
    - 将 $v=K$ 代入方程，导数项都为0，剩下 $a_0 K = b_0$。
- **结论**：
    $$v_p(t) = \frac{b_0}{a_0}$$
    - *物理意义*：这就是电路的**最终稳态值**。

---

### Page 6: 数学解法 (3) - 求齐次解
![[Lecture 14.pdf#page=6]]
**讲解：**
- **特征方程**：$s^2 + a_1 s + a_0 = 0$。
- **求根公式**：算出两个根 $s_1, s_2$。
- **两个关键参数**（务必背诵）：
    1.  **$\alpha$ (Neper frequency)**：衰减系数/阻尼系数，$\alpha = a_1/2$。决定衰减快慢。
    2.  **$\omega_0$ (Resonant frequency)**：谐振频率，$\omega_0 = \sqrt{a_0}$。决定振荡频率。
- **根的形式**：$s_{1,2} = -\alpha \pm \sqrt{\alpha^2 - \omega_0^2}$。

---

### Page 7: 情况1 - 过阻尼 (Overdamped)
![[Lecture 14.pdf#page=7]]
**讲解：**
- **条件**：$\alpha > \omega_0$ （阻力很大）。
- **特征**：根是**两个不相等的实数**。
- **响应形式**：
    $$v(t) = \text{特解} + A_1 e^{s_1 t} + A_2 e^{s_2 t}$$
    - *物理现象*：电流/电压单调地（或者稍微过冲一下）回到稳态，**不会振荡**。就像门缓慢关闭，不会由于惯性来回摆动。

---

### Page 8: 情况2 - 临界阻尼 (Critically Damped)
![[Lecture 14.pdf#page=8]]
**讲解：**
- **条件**：$\alpha = \omega_0$ （恰到好处）。
- **特征**：根是**两个相等的实数** ($s_1 = s_2 = -\alpha$)。
- **响应形式**：
    $$v(t) = \text{特解} + (A_1 + A_2 t) e^{-\alpha t}$$
    - *注意*：这里多了一个 $t$ 乘在指数项前面。
    - *物理现象*：这是**最快**回到稳态且**不发生振荡**的情况。

---

### Page 9: 情况3 - 欠阻尼 (Underdamped)
![[Lecture 14.pdf#page=9]]
**讲解：**
- **条件**：$\alpha < \omega_0$ （阻力太小，刹不住车）。
- **特征**：根是**共轭复数** ($-\alpha \pm j\beta$)。
- **响应形式**：
    $$v(t) = \text{特解} + e^{-\alpha t} [B_1 \cos(\beta t) + B_2 \sin(\beta t)]$$
    - $\beta = \sqrt{\omega_0^2 - \alpha^2}$ 是实际的振荡频率。
    - *物理现象*：电路会**振荡**！电压会在目标值上下波动，幅度越来越小，最后停在稳态值。

---

### Page 10: 串联 RLC 电路 (Series RLC)
![[Lecture 14.pdf#page=10]]
**讲解：**
- **电路模型**：R、L、C 串联。
- **微分方程推导**：利用 KVL（电压和为0）。
    - 得到的标准形式中：
    - **$\alpha = \frac{R}{2L}$** （串联电路的 $\alpha$ 公式，重点！）
    - **$\omega_0 = \frac{1}{\sqrt{LC}}$**
- **解题第一步**：算出 $\alpha$ 和 $\omega_0$，比较大小，确定是哪种阻尼。

---

### Page 11-12: 例题 8.11 (串联 - 过阻尼)
![[Lecture 14.pdf#page=11]]
![[Lecture 14.pdf#page=12]]
**讲解：**
- **步骤**：
    1.  算参数：$\alpha = 15000, \omega_0 \approx 14142$。因为 $\alpha > \omega_0$，所以是**过阻尼**。
    2.  求根：$s_1 = -10000, s_2 = -20000$。
    3.  特解：$v_p = V_s = 10V$（电容最终充到的电压）。
    4.  列通解：$v(t) = 10 + A_1 e^{-10000t} + A_2 e^{-20000t}$。
    5.  **求系数 $A_1, A_2$**（难点）：
        - 利用初值 $v(0) = V_0 = 5V$。
        - 利用初值导数 $\frac{dv(0)}{dt} = \frac{i(0)}{C} = \frac{I_0}{C}$。**切记：电容电压的导数与初始电流有关！**
- **图表**：电压缓慢上升，没有过冲振荡。

### Page 13-14: 例题 8.12 (串联 - 临界阻尼)
![[Lecture 14.pdf#page=13]]
![[Lecture 14.pdf#page=14]]
**讲解：**
- **参数变化**：电阻变小了，阻尼变小。
- **判断**：算出 $\alpha = 2000, \omega_0 = 2000$。相等，所以是**临界阻尼**。

### Page 17: 并联 RLC 电路 (Parallel RLC)
![[Lecture 14.pdf#page=17]]
**讲解：**
- **电路模型**：R、L、C 并联，通常由电流源驱动。
- **微分方程推导**：利用 KCL（电流和为0）。
- **关键区别**：
    - **$\alpha = \frac{1}{2RC}$** （并联电路的公式，与串联不同！分母是 C 不是 L）。
    - $\omega_0 = \frac{1}{\sqrt{LC}}$ （谐振频率公式不变）。
- **求解对象**：并联电路通常先解**电感电流 $i(t)$** 或 **电容电压 $v(t)$**，方程形式是一样的。

---

### Page 18-19: 例题 8.15 (并联 - 过阻尼)
![[Lecture 14.pdf#page=18]]
![[Lecture 14.pdf#page=19]]
**讲解：**
- **步骤**：
    1.  用并联公式算 $\alpha$ 和 $\omega_0$。判断为过阻尼。
    2.  特解（稳态值）：电感最后短路，所有电流 $I_s$ 都流过电感，所以 $i(\infty) = 12mA$。
    3.  列通解并求系数。
    4.  **注意初值导数**：对于电感电流，$\frac{di(0)}{dt} = \frac{v(0)}{L} = \frac{V_0}{L}$。**电感电流的导数由初始电压决定！**

### Page 25: 复杂电路例题 8.5
![[Lecture 14.pdf#page=25]]
**讲解：**
- 这个电路不是简单的 RLC 串联或并联，而是一个混合电路。
- **方法**：
    1.  化简电路：在 $t \ge 0$ 时，左边的电源和电阻对右边没影响。电路变成 R、L、C 的一个回路（实际上是串联 RLC）。
    2.  确定 $R_{total} = R_2 + R_3$。
    3.  然后就变成标准的串联 RLC 问题了。
- **初值计算**：$t<0$ 时，电感相当于短路，电容相当于开路，利用直流电路知识算出 $I_0$ 和 $V_0$。

### Page 26: 复杂电路例题 8.10
![[Lecture 14.pdf#page=26]]
**讲解：**
- 开关动作后，左边电源断开。
- 右边剩下的部分：$R_3$ 与 $R_2$ 串联吗？不，仔细看节点，它们和 L、C 形成一个闭合回路。

### Page 27: 双电容电路例题 8.19
![[Lecture 14.pdf#page=27]]
**讲解：**
- 电路里没有电感，而是有两个电容 $C_1, C_2$。
- **这依然是二阶电路**，因为有两个独立的储能元件。
- **解法**：列节点电压方程（KCL），消元得到关于 $v_2(t)$ 的二阶微分方程。
- 然后步骤同上：求特征根 -> 判断阻尼 -> 求系数。

### Page 28-30: 终极总结 (Summary)
![[Lecture 14.pdf#page=28]]
![[Lecture 14.pdf#page=30]]
**讲解：**
这三页是**考试复习的Cheat Sheet**：
1.  **通用步骤**：
    - 写出微分方程（或直接判断电路类型）。
    - 找特解（稳态值）。
    - 找齐次解（暂态值）。
    - 两者相加，代入初值求系数。
2.  **串联 vs 并联公式对比**：
    - **串联**：$\alpha = \frac{R}{2L}$, 初值导数 $\frac{dv(0)}{dt} = \frac{I_0}{C}$。
    - **并联**：$\alpha = \frac{1}{2RC}$, 初值导数 $\frac{di(0)}{dt} = \frac{V_0}{L}$。
3.  **阻尼判据**：
    - 过阻尼：$\alpha > \omega_0$
    - 临界阻尼：$\alpha = \omega_0$
    - 欠阻尼：$\alpha < \omega_0$
