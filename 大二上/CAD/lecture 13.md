**讲解：**

2.  **Step response (阶跃响应)**：突然给电路接上一个直流电源，或者突然断开，电流和电压怎么从一个状态过渡到另一个状态？

### Page 3: 核心概念引入
![[Lecture 13.pdf#page=3]]
**讲解：**
- **基本物理特性**：
    - 电容和电感是有“记忆”的元件，它们能储存能量。
    - 即使没有外部输入信号，只要它们肚子里有货（初始能量），电路就会工作，直到能量耗尽。
- **两种响应的定义**：
    1.  **自然响应 (Natural Response)**：也叫“零输入响应”或“无源响应”。
        - *通俗理解*：就像把充满气球的口松开，气球自己漏气的过程。完全靠初始储存的能量维持。
    2.  **阶跃响应 (Step Response)**：电路对直流输入信号（Step input）的反应。
        - *通俗理解*：既包含电源的作用，也包含电感初始状态的影响。

### Page 4: 自然响应的推导 (1) - 微分方程
![[Lecture 13.pdf#page=4]]
**讲解：**
- **场景**：
    - $t < 0$ 时，开关长期闭合，电感被电源 $I_S$ 充满电，初始电流 $i(0) = I_S$。
    - $t = 0$ 时，开关打开，电源断开。右边的电路变成了一个只有电感 $L$ 和电阻 $R$ 的回路（图 7.30）。
- **数学建模**：
    - 根据 KVL/KCL（基尔霍夫定律），电阻上的电压 + 电感上的电压 = 0。
    - 列出方程：$i(t)R + L\frac{di(t)}{dt} = 0$。
    - 这是一个**一阶线性微分方程**。
- **求解思路**：通过分离变量法积分，得到自然对数形式 $\ln[i(t)]$。

---

### Page 5: 自然响应的推导 (2) - 结果与功率
![[Lecture 13.pdf#page=5]]
**讲解：**
这一页给出了自然响应的**终极公式**，请务必背诵：
1.  **电流公式**：
    $$i(t) = I_0 e^{-\frac{t}{L/R}}$$
    - *含义*：电流从初始值 $I_0$ 开始，按指数规律衰减（越来越弱），直到为0。
2.  **电压公式**：
    $$v(t) = -R I_0 e^{-\frac{t}{L/R}}$$
    - *注意*：电压是负的，因为电感正在作为电源放电，电流流出端为正。
3.  **功率与能量**：电感释放能量，电阻消耗能量（发热）。

---

### Page 6: 时间常数 (Time Constant) $\tau$
![[Lecture 13.pdf#page=6]]
**讲解：**
- **定义**：**$\tau = \frac{L}{R}$** (单位：秒)
    - 这是RL电路最重要的参数，决定了电流衰减的**快慢**。
- **图表分析**：
    - 图 7.31 展示了不同 $\tau$ 值的衰减曲线。
    - $\tau$ 越大（L越大或R越小），曲线越平缓，电流衰减得越**慢**（电感“惯性”大）。
    - $\tau$ 越小，电流瞬间就没了。

---

### Page 7: 时间常数的数值意义
![[Lecture 13.pdf#page=7]]
**讲解：**
- **关键时间点**：
    - $t = \tau$ 时，电流衰减到初始值的 **36.8%** ($e^{-1}$)。
    - $t = 5\tau$ 时，电流衰减到初始值的 **0.7%**。
- **结论**：工程上认为，经过 **5个时间常数 ($5\tau$)** 后，瞬态过程结束，电路进入稳态（电感里的电放完了）。

### Page 8: 衰减速率
![[Lecture 13.pdf#page=8]]
**讲解：**
- **衰减最快的时候**：就是在 $t=0$ 的一瞬间。
- **图形含义**：

### Page 9: 解题“三部曲” (重点方法)
![[Lecture 13.pdf#page=9]]
**讲解：**
如果电路里有多个电阻，怎么算？
1.  **求初始电流 $i(0)$**：
    - 在 $t < 0$ 的稳态（开关动作前），把**电感看作短路**（导线），计算流过它的电流。
2.  **求等效电阻 $R_{eq}$**：
    - 从电感两端看进去的 **戴维宁等效电阻**。
    - *注意*：计算 $R_{eq}$ 时，要关掉所有独立电源（电压源短路，电流源开路）。
3.  **计算时间常数**：$\tau = \frac{L}{R_{eq}}$。
4.  **套公式**：$i(t) = i(0)e^{-t/\tau}$。

### Page 10: 例题 7.9 (基础题)
![[Lecture 13.pdf#page=10]]
**讲解：**
- **$t < 0$ (准备阶段)**：
    - 开关是闭合的，电感被短路。
    - 电感电流 = 电阻 $R_1$ 的电流 = $12V / 4k\Omega = 3mA$。这就是 $I_0$。
- **$t \ge 0$ (响应阶段)**：
    - 开关打开，左边电源断了。
    - 电路只剩下右边的 $L$ 和 $R_2$。
    - $R_{eq} = R_2 = 100\Omega$。
    - $\tau = L/R = 0.5H / 100\Omega = 0.005s = 5ms$。
- **结果**：$i(t) = 3e^{-200t} mA$。

### Page 15: 阶跃响应 (Step Response) - 理论 (1)
![[Lecture 13.pdf#page=15]]
**讲解：**
- **场景**：这次不仅有初始能量，电路里还保留了一个电源 $I_S$。电感最终会被这个电源充满，而不是衰减到0。
- **微分方程**：
    - 方程变成了非齐次方程：$\frac{di}{dt} + \frac{R}{L}i = \frac{R}{L}I_S$。
- **解的形式**：
    - $i(t) = I_S + A e^{-t/\tau}$。
    - 这里的 $I_S$ 是稳态值（最终值），后面一项是暂态分量。

---

### Page 16: 阶跃响应 - 理论 (2)
![[Lecture 13.pdf#page=16]]
**讲解：**
这是本章**第二个必背公式**（通用公式）：
$$i(t) = i(\infty) + [i(0) - i(\infty)] e^{-t/\tau}$$
- **$i(\infty)$**：**终值**（Final Value）。时间无穷大时，电感相当于短路，由外部电源决定的电流。
- **$i(0)$**：**初值**（Initial Value）。$t=0$ 时刻电感原本的电流。
- **$\tau$**：时间常数。
- *口诀*：**“终值 + (初值 - 终值) 乘以 e的指数”**。这个公式适用于所有一阶电路（RL和RC）！

---

### Page 17: 稳态分析
![[Lecture 13.pdf#page=17]]
**讲解：**
- 解释了为什么 $i(\infty) = I_S$。
- 当 $t \to \infty$，电流不再变化，$\frac{di}{dt} = 0$。电感电压 $L\frac{di}{dt} = 0$。
- 电压为0意味着**短路**。所以最终电流就是电流源的电流。

---

### Page 18: 阶跃响应的曲线
![[Lecture 13.pdf#page=18]]
**讲解：**
- **上升曲线**：电流从 $0$（或其他初值）慢慢爬升到终值 $I_S$。
- **数值点**：
    - $1\tau$ 时，达到终值的 **63.2%**。
    - $5\tau$ 时，达到终值的 **99.3%**（基本充满了）。
- **$R_{eq}$ 的求法**：再次强调，求 $\tau$ 时，要把独立源“杀死”（电压源短路，电流源开路），然后看电阻。

### Page 19: 例题 7.13 (套公式)
![[Lecture 13.pdf#page=19]]
**讲解：**
- **已知**：$I_S = 5mA$ (终值)，$i(0) = 0$ (初值)。
- **计算**：
    - $\tau = L/R = 3.5ms$。
    - 代入通用公式：$i(t) = 5 + (0 - 5)e^{-t/\tau} = 5(1 - e^{-t/\tau}) mA$。
- **图形**：标准的充电曲线，从0升到5mA。

### Page 20: 例题 7.14 (电压源情况)
![[Lecture 13.pdf#page=20]]
**讲解：**
- 如果电源是电压源 $V_s$ 而不是电流源，公式还是一样用！
- **终值 $i(\infty)$**：当电感短路时，电流 = $V_s / R$。
- **代入公式**：
    $$i(t) = \frac{V_s}{R} + (I_0 - \frac{V_s}{R})e^{-t/\tau}$$

### Page 21: 例题 7.15 (分段开关 - 重点难点)
![[Lecture 13.pdf#page=21]]
**讲解：**
这道题涉及两个时间段，需要分段计算。
1.  **第一阶段 ($0 \le t \le 12 \mu s$)**：
    - 开关1打开。电路处于自然响应（衰减）。
    - 算出 $\tau_1$。
    - 关键：算出 $t=12 \mu s$ 这一刻的电流值 $i(12\mu s)$，它是下一阶段的**初值**。
2.  **第二阶段 ($t > 12 \mu s$)**：
    - 开关2闭合。电路结构改变，接入了新电阻和电源。
    - 算出新的 $\tau_2$ 和新的终值 $i(\infty)$。
    - 将 $i(12\mu s)$ 作为初值代入通用公式，注意时间变量要写成 $(t - 12\times 10^{-6})$。

### Page 23: 例题 7.17 (多电源复杂电路)
![[Lecture 13.pdf#page=23]]
**讲解：**
- **$t < 0$**：电路里既有电流源又有电压源。
    - 使用**叠加定理**或者**节点电压法**算出电感里的初始电流 $i(0) = 32mA$。

### Page 24: 例题 7.18 (含受控源)
![[Lecture 13.pdf#page=24]]
**讲解：**
- **难点**：电路里有一个菱形符号的受控电压源 $2 di(t)/dt$。
- **方法**：含受控源的电路，通常**不能直接简单求 $R_{eq}$**。
- **解法**：必须列 KVL 微分方程。
    - $-9 + 3i + 5\frac{di}{dt} + 2\frac{di}{dt} + 6i = 0$ (假设右边是闭合回路)。
    - 整理方程：$\frac{di}{dt} + 0.6i = 1.8$。
    - 从方程系数直接读出 $\tau$ 和终值。
    - 系数 $0.6$ 对应 $R/L_{total}$，常数项 $1.8$ 对应 $V/L_{total}$。

### Page 25: 总结 (Summary)
![[Lecture 13.pdf#page=25]]
**讲解：**
本页汇总了解决RL电路问题的两个黄金公式：
1.  **无源（自然响应）**：$i(t) = i(0)e^{-t/\tau}$。
2.  **有源（阶跃响应）**：$i(t) = i(\infty) + (i(0) - i(\infty))e^{-t/\tau}$。
- 只要记住第二个公式（通用公式），令 $i(\infty)=0$ 就变成了第一个公式。所以**只背第二个就够了**。
-
