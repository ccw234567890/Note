## 第3页：电感器基础
![[Lecture 11_Hainan-Ali.pdf#page=3]]
本页介绍了电感器的基本概念和物理结构。

* **电感器 (Inductor)** 是一种**无源元件**，能够以**磁场**的形式**储存能量**。
* **物理结构**：一个简单的电感器通常由一根导线绕成螺旋状的**螺线管**构成。
* **电路符号**：图6.26展示了电感器的电路符号。
* **磁芯**：线圈内部可以什么都不填（空芯），也可以填充铁磁性材料（如铁芯、铁氧体磁芯）来增强电感。
* **工作原理**：当电流流过电感器时，会产生一个磁场。根据法拉第定律，**变化的磁场**会产生**感应电动势**。
* **电感 (Inductance)**：是衡量电感器在给定电流变化率下产生磁通量（或感应电动势）能力的物理量。
* **单位**：电感的单位是**亨利 (H)**，以纪念美国科学家约瑟夫·亨利 (Joseph Henry)，他在1832年独立于法拉第发现了自感现象。
* **储能**：储存在电感器中的能量大小取决于其电感值和流过的电流。

## 第5页：电感器的电流-电压关系
![[Lecture 11_Hainan-Ali.pdf#page=5]]
本页推导了电感器两端的电压与其内部电流的关系。

* **楞次定律的应用**：当流过电感的电流 $i(t)$ 增加时（$di(t)/dt > 0$），电感器会产生一个感应电动势，这个电动势会产生一个**感应电流 $i_{induced}(t)$**，其方向与原电流 $i(t)$ **相反**，以阻碍原电流的增加。
* **电压极性**：为了产生这个反向的感应电流，感应电压的极性（上正下负）会阻碍原电流的流入。
* **电压-电流关系**：电感器两端的电压 $v(t)$ 与流过它的电流 $i(t)$ 的**变化率**成正比：
    $$v(t)=L\frac{di(t)}{dt}$$
* **趣味知识（电感特性）**：电感**阻碍电流突变**的特性使其在工程中非常有用。例如，在汽车的点火线圈（一个电感器）中，当电流被突然切断时（$di/dt$ 极大），会产生一个高达2万到5万伏特的瞬时高压，足以击穿火花塞的间隙，点燃燃料。

## 第6页：电感器的电压-电流关系与储能
![[Lecture 11_Hainan-Ali.pdf#page=6]]
本页推导了电感器的电流和能量公式。

* **电压-电流关系**：对上一页的电压公式进行积分，可以得到电流的表达式：
    $$i(t)=\frac{1}{L}\int_{-\infty}^{\prime}v(\lambda)d\lambda = i(0)+\frac{1}{L}\int_{0}^{\prime}v(\lambda)d\lambda$$
    其中 $i(0)$ 是电感器在 t=0 时刻的初始电流。
* **瞬时功率**：$p(t)=v(t)i(t) = [L\frac{di(t)}{dt}]i(t)$
* **储存的能量**：对功率进行积分，得到储存在电感器中的能量 $w(t)$：
    $$w(t)=\int_{-\infty}^{\prime}p(\lambda)d\lambda = \frac{1}{2}Li^{2}(t)$$
* 如果电流为恒定值 I（直流，di/dt = 0），则储存的能量为 $W=0.5LI^{2}$。

* **给定**：$L = 100 mH$，电流 $i(t)$ 随时间呈三角波形（如图6.35）。
* **a. 求解电压 v(t)** (第8页)：

## 第11页：正弦输入：相位关系
![[Lecture 11_Hainan-Ali.pdf#page=11]]
本页分析了电感器在正弦电流输入下的一个关键特性。

* **情景**：一个正弦电流 $i(t) = \cos(2\pi 10t) A$ 施加在电感器上。
* **推导电压**：
    * $v(t) = L \frac{di(t)}{dt} = L \frac{d}{dt}[\cos(2\pi 10t)]$
    * $v(t) = -L (2\pi 10) \sin(2\pi 10t)$
    * 利用三角恒等式 $-\sin(\theta) = \cos(\theta + 90^\circ)$，可得：
        $v(t) = L (2\pi 10) \cos(2\pi 10t + 90^\circ)$
* **结论**：在纯电感电路中，**电压 $v(t)$ 的相位超前电流 $i(t)$ 90度（或 $\pi/2$ 弧度）**。（反过来说，电流滞后电压90度）。
* 如图6.44所示，电流波形（实线）比电压波形（虚线）晚了 $T/4$ 个周期（即90度）达到峰值。

## 第12页：正弦输入：频率依赖性
![[Lecture 11_Hainan-Ali.pdf#page=12]]
本页分析了电感器对不同频率信号的响应。

* **电压幅值**：从上一页的推导 $v(t) = -LI_m (2\pi f) \sin(2\pi ft)$ 可知，电压的幅值 $V_m = (2\pi f L) I_m$。
* **结论**：电压的幅值与输入电流的**频率 f 成正比**。
* **特性**：
    * **高频**：频率 f 很高时，电压幅值很大。电感器近似于**开路 (open circuit)**（对电流阻碍很大）。
    * **低频**：频率 f 很低时，电压幅值很小。
    * **直流 (DC, f=0)**：频率 f=0 时，电压为零（在稳态下）。电感器近似于**短路 (short circuit)**（对电流无阻碍）。
* 图6.46展示了在相同电流幅值下，不同频率 (1Hz, 5Hz, 10Hz) 对应的电压波形。

## 第13页：电感器的串联
![[Lecture 11_Hainan-Ali.pdf#page=13]]
本页推导了电感器串联时的等效电感公式。

* **串联特点** (图6.47a)：
    * 流过每个电感器的**电流 $i(t)$ 相同**。
    * 总电压 $v(t)$ 等于每个电感器上的电压之和：$v(t) = v_1(t) + v_2(t) + ... + v_n(t)$。
* **推导**：
    * 将 $v_k(t) = L_k \frac{di(t)}{dt}$ 代入电压求和式。
    * $v(t) = (L_1 + L_2 + ... + L_n) \frac{di(t)}{dt}$。
    * 将其与等效电感的公式 $v(t) = L_{eq} \frac{di(t)}{dt}$ 进行比较。
* **串联等效电感公式**：
    $$L_{eq}=L_{1}+L_{2}+...+L_{n}$$
* **重要特性**：电感串联的等效电感公式，形式上与**电阻串联**的公式相同。

## 第14页与第15页：电感器的并联
![[Lecture 11_Hainan-Ali.pdf#page=14]]
![[Lecture 11_Hainan-Ali.pdf#page=15]]
这两页推导了电感器并联时的等效电感公式。

* **并联特点** (图6.48a)：
    * 每个电感器两端的**电压 $v(t)$ 相同**。
    * 总电流 $i(t)$ 等于流过每个电感器的电流之和：$i(t) = i_1(t) + i_2(t) + ... + i_n(t)$。
* **推导** (第14页)：
    * 将 $i_k(t) = \frac{1}{L_k} \int v(\lambda)d\lambda$ 代入电流求和式。
    * $i(t) = (\frac{1}{L_1} + \frac{1}{L_2} + ... + \frac{1}{L_n}) \int v(\lambda)d\lambda$。
    * 将其与等效电感的公式 $i(t) = \frac{1}{L_{eq}} \int v(\lambda)d\lambda$ 进行比较。
* **并联等效电感公式**：
    $$\frac{1}{L_{eq}}=\frac{1}{L_{1}}+\frac{1}{L_{2}}+...+\frac{1}{L_{n}}$$
* **重要特性** (第15页)：
    * 电感器并联的等效电感公式，形式上与**电阻并联**的公式相同。

## 第16页与第17页：示例 6.10（串并联混合计算）
![[Lecture 11_Hainan-Ali.pdf#page=16]]
![[Lecture 11_Hainan-Ali.pdf#page=17]]
这两页通过一个复杂的串并联混合电路，演示了如何逐步求解总等效电感。

* **目标**：求解图6.49中a、b两点间的等效电感 $L_{eq}$。
* **求解步骤** (第17页)：
    1.  **$L_4, L_5, L_6$ 并联**：先计算这三个并联电感的等效电感 $L_a$。
    2.  **$L_3$ 和 $L_a$ 串联**：计算 $L_b = L_3 + L_a$。
    3.  **$L_2$ 和 $L_b$ 并联**：计算 $L_c = L_2 || L_b$。
    4.  **$L_1$ 和 $L_c$ 串联**：计算最终的等效电感 $L_{eq} = L_1 + L_c$。
    * 最终结果为 $L_{eq} = 5 mH$。

这几页（内容有重复）演示了如何分别简化一个LC混合电路的电感部分和电容部分，以得到一个最终的等效并联LC电路。
