## 第3页：简介
![[Lecture 12_Hainan-Ali.pdf#page=3]]
本页介绍了RC电路的两种基本响应类型。

* 当电容器或电感器存有初始能量时，即使没有外部输入信号，电路也会对这些初始能量做出响应，直到能量耗尽。
* **自然响应 (Natural Response)**：电路**仅由初始能量**引起的响应。也称为瞬态响应、零输入响应或无源响应。
* **阶跃响应 (Step Response)**：电路对一个**直流输入信号**（阶跃输入）的响应。
* 阶跃响应包含了由初始能量和外部输入共同引起的响应。

## 第4页与第5页：RC电路的自然响应
![[Lecture 12_Hainan-Ali.pdf#page=4]]
![[Lecture 12_Hainan-Ali.pdf#page=5]]
这两页推导了RC电路自然响应（即放电过程）的数学表达式。

### 情景
* 如图7.1所示，开关在 t=0 之前长时间处于位置 a，电容器被电压源 $V_S$ 充电至 $v(0) = V_0 = V_S$。
* 在 t=0 时刻，开关切换到位置 b，电容器开始通过电阻 R 放电（如图7.2所示）。

### 推导过程 (第4页)
1.  **列方程 (t ≥ 0)**：在图7.2的RC回路中，根据基尔霍夫电流定律 (KCL)，流出节点1的总电流为零：
    $$C\frac{dv(t)}{dt} + \frac{v(t)}{R} = 0$$
2.  **分离变量**：将方程整理为 $\frac{dv(t)}{v(t)} = -\frac{1}{RC}dt$。
3.  **两边积分**：$\ln[v(t)] = -\frac{t}{RC} + K$ (其中 K 是积分常数)。
4.  **取指数**：$v(t) = e^{-\frac{t}{RC} + K} = e^{K} e^{-t/RC} = A e^{-t/RC}$。

### 结论 (第5页)
5.  **求解常数 A**：利用初始条件 $t=0, v(0)=V_0$。
    $v(0) = A e^0 = A$，因此 $A = V_0$。
* **电压响应**：$v(t) = V_0 e^{-t/RC}$ (t ≥ 0)。电压以指数形式衰减。
* **电流响应**：$i(t) = C\frac{dv(t)}{dt} = C V_0 (-\frac{1}{RC}) e^{-t/RC} = -\frac{V_0}{R} e^{-t/RC}$ (t ≥ 0)。电流也以指数形式衰减，负号表示电流流出电容正极。
* **功率与能量**：
    * **电容瞬时功率**：$p_C(t) = v(t)i(t) = -\frac{V_0^2}{R}e^{-2t/RC}$ (负号表示电容在释放能量)。
    * **电阻吸收的能量**：电阻消耗的总能量 $w_R(\infty)$ 最终等于电容器初始储存的全部能量 $0.5CV_0^2$。

## 第8页：RC电路的时间常数
![[Lecture 12_Hainan-Ali.pdf#page=8]]
本页详细解释了**时间常数 $\tau$** 的物理意义。

* **定义**：时间常数 $\tau = RC$，单位是秒。
* **物理意义**：$\tau$ 是控制电容器放电快慢的参数。
    * 响应公式可以写为：$v(t) = V_0 e^{-t/\tau}$。
    * 如图7.4所示，**$\tau$ 越大，电容器放电越慢**。
* $\tau$ 控制着电容器储存的电荷释放的速率。

## 第9页：时间常数 $\tau$ 的定量分析
![[Lecture 12_Hainan-Ali.pdf#page=9]]
本页给出了时间常数的定量含义。

* **在 $t = \tau$ 时**：
    * $v(\tau) = V_0 e^{-1} \approx 0.36788 V_0$。
    * 也就是说，经过**一个时间常数**，电容电压会衰减到其初始值的**36.8%** 左右。
* **在 $t = 5\tau$ 时**：
    * $v(5\tau) = V_0 e^{-5} \approx 0.006738 V_0$。

## 第10页：时间常数 $\tau$ 的切线意义
![[Lecture 12_Hainan-Ali.pdf#page=10]]
本页通过图表展示了时间常数的另一种几何意义。

* **初始衰减率**：在 $t=0$ 时刻，电压的衰减率（斜率）最大，为 $-V_0/\tau$。
* **切线意义**：如果电压始终保持这个**初始衰减率**（如图7.6中的虚线 $v_1(t)$ 所示），那么它将在**一个时间常数 $\tau$** 后衰减到零。
* **实际情况**：实际的电压 $v(t)$（实线）衰减率会不断减小，所以衰减过程更慢。
* **5$\tau$ 准则**：经过**五个时间常数**（$t=5\tau$），电容电压已衰减到初始值的**1%以下**。在工程实践中，通常认为此时瞬态响应已经结束。

## 第11页：如何求解复杂电路的时间常数
![[Lecture 12_Hainan-Ali.pdf#page=11]]
本页给出了求解更复杂RC电路自然响应的通用方法。

* **步骤**：
    1.  **求初始电压 $v(0)$**：分析 t=0 时刻电容器两端的电压。
    2.  **求等效电阻 $R_{eq}$**：对于包含多个电阻的电路，需要找到从**电容器两端**看进去的**戴维南等效电阻 $R_{eq}$**。
    3.  **求时间常数 $\tau$**：$\tau = R_{eq}C$。
    4.  **写出响应**：$v(t) = v(0) e^{-t/\tau}$。

## 第12页至第14页：示例 7.2（含多个电阻的自然响应）
![[Lecture 12_Hainan-Ali.pdf#page=12]]
![[Lecture 12_Hainan-Ali.pdf#page=13]]
![[Lecture 12_Hainan-Ali.pdf#page=14]]
这几页演示了如何应用上述通用方法求解复杂RC电路的自然响应。

* **情景**：在图7.7中，开关SW在 t=0 前长时间闭合，在 t=0 时刻断开。
* **1. 求解初始电压 $v(0)$** (第13页)：
    * 在 t < 0 时，电路处于直流稳态，电容器相当于**开路**。
    * $v(0)$ 等于 $R_2$ 两端的电压。$R_3$ 和 $R_4$ 串联 ($30 k\Omega$)，再与 $R_2$ ($15 k\Omega$) 并联。
    * $R_{eq\_t<0} = 15k\Omega || 30k\Omega = 10 k\Omega$。
    * $v(0)$ 通过 $V_S$、$R_1$ 和 $R_{eq\_t<0}$ 构成的分压电路得到：$v(0) = 15V \times \frac{10k\Omega}{5k\Omega + 10k\Omega} = 10V$。
* **2. 求解等效电阻 $R_{eq}$ (t ≥ 0)** (第13页)：
    * 在 t ≥ 0 时，开关断开，左侧电源被切断。
    * 从电容器两端看进去，$R_2$ 与 ($R_3+R_4$) 处于**并联**关系。
    * $R_{eq} = R_2 || (R_3 + R_4) = 15k\Omega || (10k\Omega + 20k\Omega) = 10 k\Omega$。
* **3. 求解时间常数 $\tau$** (第13页)：
    * $\tau = R_{eq}C = 10k\Omega \times 0.5\mu F = 5 ms$。
* **4. 写出响应** (第14页)：
    * $v(t) = v(0)e^{-t/\tau} = 10e^{-t/5ms} V = 10e^{-200t} V$ (t ≥ 0)。
    * $i(t) = C \frac{dv(t)}{dt} = -1e^{-200t} mA$ (t ≥ 0)。
* **波形图 (第14页)**：绘制了电压和电流的衰减曲线。

## 第15页：示例 7.3
![[Lecture 12_Hainan-Ali.pdf#page=15]]
本页是另一个自然响应的例子。

* **情景**：在图7.12中，开关SW在 t=0 前长时间闭合，在 t=0 时刻断开。
* **1. 求解 $v(0)$**：
    * t < 0 时，电路稳态，电容开路。
    * $v(0)$ 等于 $R_3$ 两端的电压。$I_s$ 的电流在 $R_1$ 和 $R_2+R_3$ 之间分流。
    * 根据**分流定律**，流过 $R_3$ 的电流为 $I_3 = 5mA \times \frac{R_1}{R_1+(R_2+R_3)} = 5mA \times \frac{3k\Omega}{3k\Omega + (1k\Omega+1k\Omega)} = 3mA$。(注意：此步计算有误，应为 $1k\Omega+3k\Omega$，但此处我们按讲义的 $R_3=1k\Omega$ 计算)
    * $v(0) = I_3 \times R_3 = 3mA \times 1k\Omega = 3V$。
* **2. 求解 $R_{eq}$** (t ≥ 0)：
    * t ≥ 0 时，开关断开，左侧电流源被切断。
    * 从电容器两端看进去，$R_3$ 和 $R_4$ 处于**串联**关系。
    * $R_{eq} = R_3 + R_4 = 1k\Omega + 3k\Omega = 4k\Omega$。
* **3. 求解 $\tau$**：
    * $\tau = R_{eq}C = 4k\Omega \times 0.5\mu F = 2 ms$。
* **4. 写出响应**：
    * $v(t) = v(0)e^{-t/\tau} = 3e^{-t/2ms} V = 3e^{-500t} V$ (t ≥ 0)。

## 第16页至第18页：RC电路的阶跃响应
![[Lecture 12_Hainan-Ali.pdf#page=16]]
![[Lecture 12_Hainan-Ali.pdf#page=17]]
![[Lecture 12_Hainan-Ali.pdf#page=18]]
这几页推导了RC电路在直流电压源（阶跃输入）作用下的响应（即充电过程）。

### 情景 (第16页)
* 如图7.15所示，一个RC串联电路在 t=0 时刻闭合开关。电容器的初始电压为 $v(0) = V_0$。

### 推导过程
1.  **列方程 (t ≥ 0)** (第16页)：根据基尔霍夫电压定律 (KVL)：$-V_S + Ri(t) + v(t) = 0$。
2.  **代入 i(t)** (第16页)：将 $i(t) = C\frac{dv(t)}{dt}$ 代入，得到一阶线性微分方程：
    $$RC\frac{dv(t)}{dt} + v(t) = V_S$$
3.  **解微分方程 (第17页)**：
    * 这可以通过分离变量法求解。
    * 积分后得到 $\ln|v(t) - V_S| = -\frac{t}{RC} + K$。
    * 取指数得到 $v(t) = V_S + A e^{-t/RC}$。
4.  **求解常数 A** (第17页)：利用初始条件 $t=0, v(t)=V_0$。
    $v(0) = V_S + A e^0 \Rightarrow V_0 = V_S + A \Rightarrow A = V_0 - V_S$。

### 结论 (第18页)
* **电压响应**：将 A 代回，得到RC电路阶跃响应的通用公式：
    $$v(t) = V_S + (V_0 - V_S) e^{-t/RC}$$
* **三要素公式**：上式可以写成一个更通用的形式：
    $$v(t) = v(\infty) + [v(0) - v(\infty)]e^{-t/\tau}$$
    其中：
    * $v(\infty) = V_S$ 是电路的**最终稳态值**（t→∞时，电容开路，电压等于 $V_S$）。
    * $v(0) = V_0$ 是电路的**初始值**。
    * $\tau = RC$ 是电路的**时间常数**。

## 第23页：示例 7.6（分段响应）
![[Lecture 12_Hainan-Ali.pdf#page=23]]
本页分析了一个包含两个开关、在不同时间点动作的RC电路。

* **情景**：图7.22中，SW1在 t=0 时断开，SW2在 t=0.5ms 时闭合。
* **分析 (t < 0)**：SW1闭合，SW2断开。电路稳态，电容开路。
    * $v(0) = 10V \times \frac{4k\Omega}{6k\Omega + 4k\Omega} = 4V$。
* **分析 (0 ≤ t < 0.5ms)**：SW1断开，SW2断开。电容通过 $R_2$ 放电。
    * $R_{eq1} = R_2 = 4k\Omega$。
    * $\tau_1 = R_{eq1}C = 4k\Omega \times 0.25\mu F = 1 ms$。
    * $v(t) = v(0)e^{-t/\tau_1} = 4e^{-1000t} V$。
* **在 t = 0.5ms 时**：$v(0.5ms) = 4e^{-0.5} \approx 2.4261V$。
* **分析 (t ≥ 0.5ms)**：SW1断开，SW2闭合。
    * 电路变为一个有源电路，需要求新的**稳态值 $v(\infty)$** 和**等效电阻 $R_{eq2}$**。
    * $v(\infty) = 20V \times \frac{R_2}{R_3 + R_2} = 20V \times \frac{4k\Omega}{6k\Omega + 4k\Omega} = 8V$。
    * $R_{eq2} = R_2 || R_3 = 4k\Omega || 6k\Omega = 2.4 k\Omega$。
    * $\tau_2 = R_{eq2}C = 2.4k\Omega \times 0.25\mu F = 0.6 ms$。
* **最终响应 (t ≥ 0.5ms)**：
    * $v(t) = v(\infty) + [v(0.5ms) - v(\infty)]e^{-(t-0.5ms)/\tau_2}$
    * $v(t) = 8 + (2.4261 - 8)e^{-(t-0.5ms)/0.6ms} V$
    * $v(t) = [8 - 5.5739e^{-1666.67(t-0.0005)}] V$。
* 图7.23展示了电压 $v(t)$ 在 t=0.5ms 处发生转折的完整波形。

## 第24页：示例 7.7
![[Lecture 12_Hainan-Ali.pdf#page=24]]
本页是另一个分段响应的例子。

* **情景**：图7.25中，SW1在 t=0 时断开，SW2在 t=0 时闭合。
* **1. 求解 $v(0)$ (t < 0)**：SW1闭合，SW2断开。电容开路。
    * $v(0) = 5V \times \frac{R_2}{R_1 + R_2} = 5V \times \frac{20k\Omega}{30k\Omega + 20k\Omega} = 2V$。
* **2. 求解 $v(\infty)$ (t → ∞)**：SW1断开，SW2闭合。电路达到新稳态，电容开路。

    * 从电容 C 两端（节点a和c）看进去。独立源置零（$V_S$ 短路，$I_S$ 开路）。

* **4. 求解 $\tau$**：
    * $\tau = R_{eq}C = 20k\Omega \times 0.02\mu F = 0.4 ms$。
* **5. 写出响应**：
    * $v(t) = v(\infty) + [v(0) - v(\infty)]e^{-t/\tau} = 4 + (2 - 4)e^{-t/0.4ms} = [4 - 2e^{-2500t}] V$ (t ≥ 0)。

## 第25页：示例 7.8（含受控源）
![[Lecture 12_Hainan-Ali.pdf#page=25]]
本页分析了一个包含受控源的RC电路。

* **情景**：图7.27中，初始电压 $v(0) = 0V$。
* **分析**：求解这类电路（含受控源）的响应，最有效的方法是先求出电容两端的**戴维南等效电路**。
    1.  **求 $v(\infty)$ (即 $V_{th}$)**：断开电容，计算a、b两端的开路电压。
    2.  **求 $R_{eq}$ (即 $R_{th}$)**：独立源置零（0.5V电压源短路），在a、b端加一个测试源来求解等效电阻。
* **讲义中的快捷方法**：
    * 讲义没有使用戴维南定理，而是直接对节点b列出了KCL方程：
        $$0.01v_a + \frac{v(t)}{R_3} + C\frac{dv(t)}{dt} = 0$$
    * 控制电压 $v_a$ 是一个恒定值：$v_a = 0.5V \times \frac{2k\Omega}{3k\Omega + 2k\Omega} = 0.2V$。
    * 方程变为：$\frac{dv(t)}{dt} + \frac{1}{R_3 C}v(t) = -\frac{0.01v_a}{C}$。
    * **最终稳态值 $v(\infty)$**：当 $dv/dt = 0$ 时，$v(\infty) = -0.01 v_a R_3 = -0.01 \times (0.2V) \times (1k\Omega) = -2V$。
    * **时间常数 $\tau$**：$\tau = R_3 C = 1k\Omega \times 1\mu F = 1 ms$。
    * **初始值 $v(0)$**：$v(0) = 0V$。
* **电压响应**：
    * $v(t) = v(\infty) + [v(0) - v(\infty)]e^{-t/\tau} = -2 + (0 - (-2))e^{-t/1ms} = [-2 + 2e^{-1000t}] V$ (t ≥ 0)。

## 第26页：总结
![[Lecture 12_Hainan-Ali.pdf#page=26]]
本页总结了RC电路的两种响应。

* **自然响应 (无源)**：
    * $v(t) = v(0)e^{-t/\tau}$
    * $\tau = RC$ (或 $R_{eq}C$)
* **阶跃响应 (有源)**：
    * $v(t) = v(\infty) + [v(0) - v(\infty)]e^{-t/\tau}$
    * $v(\infty)$ 是最终稳态值， $v(0)$ 是初始值。
    * $\tau = R_{eq}C$ (其中 $R_{eq}$ 是从电容两端看进去的戴维南等效电阻)。
