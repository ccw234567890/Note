## 第3页：电容器：基本原理
![[Lecture 10_Hainan- Ali.pdf#page=3]]
本页介绍了电容器的基本概念。

* 电容器是一种**无源元件 (passive element)**，能够**储存和释放能量**。
* 一个平行板电容器由两个金属极板和填充在中间的**介电材料 (dielectric material)** 组成。
* 正极板储存正电荷，负极板储存负电荷。
* **电容 (Capacitance)**：衡量电容器在给定电压下储存电荷的能力。
* 能量以**电场**的形式储存在两极板之间。
* 储存的能量大小取决于电容值和极板间的电压。
* 图6.1展示了电容器的电路符号。

## 第4页：电容的定义
![[Lecture 10_Hainan- Ali.pdf#page=4]]
本页给出了电容的数学定义。

* 当电容器连接到电压源（如电池）时，电荷会在极板上积累。
* 极板上储存的电荷量 Q（库仑）与极板间的电压 V（伏特）成正比。
* **电容 (C)** 就是这个比例常数，定义为储存的电荷与电势差（电压）之比：
    $$Q=CV \Rightarrow C=Q/V$$
* 电容的单位是**法拉 (F)**。
* 如果电压是随时间变化的 $v(t)$，那么电荷也是随时间变化的 $Q(t) = C \cdot v(t)$。

## 第5页：电容器的电流-电压关系
![[Lecture 10_Hainan- Ali.pdf#page=5]]
本页推导了流过电容器的电流与其两端电压的关系。

* 电流 $i(t)$ 被定义为电荷 $Q(t)$ 随时间的变化率：
    $$i(t)=\frac{dQ(t)}{dt}$$
* 将 $Q(t)=Cv(t)$ 代入上式，得到电容器的**电流-电压关系**：
    $$i(t)=C\frac{dv(t)}{dt}$$
* **结论**：流过电容器的电流正比于其两端**电压随时间的变化率**（即电压对时间的导数）。
    * 如果电压恒定不变（直流，dv/dt = 0），则电流为零。
    * 电压变化越快，电流越大。

## 第6页：电容器的电压-电流关系与储能
![[Lecture 10_Hainan- Ali.pdf#page=6]]
本页推导了电容器的电压和能量公式。

* **电压-电流关系**：对上一页的电流公式进行积分，可以得到电压的表达式：
    $$v(t)=\frac{1}{C}\int_{-\infty}^{\prime}i(\lambda)d\lambda = v(0)+\frac{1}{C}\int_{0}^{\prime}i(\lambda)d\lambda$$
    其中 $v(0)$ 是电容器在 t=0 时刻的初始电压。
* **瞬时功率**：$p(t)=v(t)i(t) = v(t)C\frac{dv(t)}{dt}$
* **储存的能量**：对功率进行积分，得到储存在电容器中的能量 $w(t)$：
    $$w(t)=\int_{-\infty}^{\prime}p(\lambda)d\lambda = \frac{1}{2}Cv^{2}(t)$$
* 如果电压为恒定值 V，则储存的能量为 $w=0.5CV^{2}$。

这几页通过一个实例，演示了如何根据给定的分段线性电压波形，求解电容器的电流、功率和能量。

* **给定**：$C = 100 \mu F$，电压 $v(t)$ 随时间呈三角波形（如图6.6）。
* **a. 求解电流 i(t)** (第8页)：
    * 在每个时间段内，电压 $v(t)$ 都是线性的，其导数 $dv(t)/dt$ 是一个常数。
    * 应用公式 $i(t)=C \frac{dv(t)}{dt}$：
        * 0 ≤ t < 1s：$dv/dt = 50$，所以 $i(t) = 100\mu F \times 50 = 5 mA$。
        * 1 ≤ t < 3s：$dv/dt = -50$，所以 $i(t) = 100\mu F \times (-50) = -5 mA$。
        * 3 ≤ t < 4s：$dv/dt = 50$，所以 $i(t) = 100\mu F \times 50 = 5 mA$。

    * 应用公式 $p(t) = v(t)i(t)$，将分段的 $v(t)$ 和 $i(t)$ 相乘，得到分段的功率函数。
* **c. 求解能量 w(t)** (第9页)：
    * 应用公式 $w(t) = \frac{1}{2}Cv^{2}(t)$，将分段的 $v(t)$ 代入。
    * 能量 $w(t)$ 在 $t=1$s 和 $t=3$s 时达到峰值，在 $t=2$s 和 $t=4$s 时降为零。

## 第13页与第14页：正弦输入：相位关系
![[Lecture 10_Hainan- Ali.pdf#page=13]]
![[Lecture 10_Hainan- Ali.pdf#page=14]]
这两页（内容有重复）分析了电容器在正弦电压输入下的一个关键特性。

* **情景**：一个正弦电压 $v(t) = V_m \cos(2\pi ft)$ 施加在电容器上。
* **推导电流**：
    * $i(t) = C \frac{dv(t)}{dt} = C \frac{d}{dt}[V_m \cos(2\pi ft)]$
    * $i(t) = -C V_m (2\pi f) \sin(2\pi ft)$
    * 利用三角恒等式 $-\sin(\theta) = \cos(\theta + 90^\circ)$，可得：
        $i(t) = (2\pi f C V_m) \cos(2\pi ft + 90^\circ)$
* **结论**：在纯电容电路中，**电流 $i(t)$ 的相位超前电压 $v(t)$ 90度（或 $\pi/2$ 弧度）**。
* 如图6.16所示，电流波形（虚线）比电压波形（实线）提前了 $T/4$ 个周期（即90度）达到峰值。

## 第15页：正弦输入：频率依赖性
![[Lecture 10_Hainan- Ali.pdf#page=15]]
本页分析了电容器对不同频率信号的响应。

* **电流幅值**：从上一页的推导 $i(t) = -CV_{m}(2\pi f) \sin(2\pi ft)$ 可知，电流的幅值 $I_m = (2\pi f C) V_m$。
* **结论**：电流的幅值与输入电压的**频率 f 成正比**。
* **特性**：
    * **高频**：频率 f 很高时，电流幅值很大。电容器近似于**短路 (short circuit)**。
    * **低频**：频率 f 很低时，电流幅值很小。
    * **直流 (DC, f=0)**：频率 f=0 时，电流为零（在稳态下）。电容器近似于**开路 (open circuit)**。
* 图6.18展示了在相同电压幅值下，不同频率 (1Hz, 5Hz, 10Hz) 对应的电流波形。

## 第16页至第18页：电容器的串联
![[Lecture 10_Hainan- Ali.pdf#page=16]]
![[Lecture 10_Hainan- Ali.pdf#page=17]]
![[Lecture 10_Hainan- Ali.pdf#page=18]]
这几页推导了电容器串联时的等效电容公式。

* **串联特点** (图6.19a)：
    * 流过每个电容器的**电流 $i(t)$ 相同**。
    * 总电压 $v(t)$ 等于每个电容器上的电压之和：$v(t) = v_1(t) + v_2(t) + ... + v_n(t)$。
* **推导** (第16页)：
    * 将 $v_k(t) = \frac{1}{C_k} \int i(\lambda)d\lambda$ 代入电压求和式。
    * $v(t) = (\frac{1}{C_1} + \frac{1}{C_2} + ... + \frac{1}{C_n}) \int i(\lambda)d\lambda$。
    * 将其与等效电容的公式 $v(t) = \frac{1}{C_{eq}} \int i(\lambda)d\lambda$ 进行比较。
* **串联等效电容公式**：
    $$\frac{1}{C_{eq}}=\frac{1}{C_{1}}+\frac{1}{C_{2}}+...+\frac{1}{C_{n}}$$
* **重要特性** (第17-18页)：
    * 电容器串联的等效电容公式，形式上与**电阻并联**的公式相同。
    * 串联后的总电容**小于**任何一个单独的电容值。

## 第19页：电容器的并联
![[Lecture 10_Hainan- Ali.pdf#page=19]]
本页推导了电容器并联时的等效电容公式。

* **并联特点** (图6.20a)：
    * 每个电容器两端的**电压 $v(t)$ 相同**。
    * 总电流 $i(t)$ 等于流过每个电容器的电流之和：$i(t) = i_1(t) + i_2(t) + ... + i_n(t)$。
* **推导**：
    * 将 $i_k(t) = C_k \frac{dv(t)}{dt}$ 代入电流求和式。
    * $i(t) = (C_1 + C_2 + ... + C_n) \frac{dv(t)}{dt}$。
    * 将其与等效电容的公式 $i(t) = C_{eq} \frac{dv(t)}{dt}$ 进行比较。
* **并联等效电容公式**：
    $$C_{eq}=C_{1}+C_{2}+...+C_{n}$$
* **重要特性**（参见第26页）：
    * 电容器并联的等效电容公式，形式上与**电阻串联**的公式相同。
    * 并联后的总电容**大于**任何一个单独的电容值。

## 第20页与第21页：示例 6.4（串并联混合计算）
![[Lecture 10_Hainan- Ali.pdf#page=20]]
![[Lecture 10_Hainan- Ali.pdf#page=21]]
这两页通过一个复杂的串并联混合电路，演示了如何逐步求解总等效电容。

* **目标**：求解图6.21中a、b两点间的等效电容 $C_{eq}$。
* **求解步骤** (第21页)：
    1.  **$C_6$ 和 $C_7$ 串联**：$C_8 = (C_6 \times C_7) / (C_6 + C_7) = 1.5 \mu F$。
    2.  **$C_4$ 和 $C_5$ 串联**：$C_9 = (C_4 \times C_5) / (C_4 + C_5) = 2.1818 \mu F$。
    3.  **$C_8$ 和 $C_9$ 并联**：$C_{10} = C_8 + C_9 = 3.6818 \mu F$。
    4.  **$C_2$、$C_3$ 和 $C_{10}$ 串联**：$1/C_{11} = 1/C_2 + 1/C_3 + 1/C_{10}$，解得 $C_{11} = 1.5667 \mu F$。
    5.  **$C_1$ 和 $C_{11}$ 并联**：$C_{eq} = C_1 + C_{11} = 3 \mu F + 1.5667 \mu F = 4.5667 \mu F$。

## 第24页至第26页：总结
![[Lecture 10_Hainan- Ali.pdf#page=24]]
![[Lecture 10_Hainan- Ali.pdf#page=25]]
![[Lecture 10_Hainan- Ali.pdf#page=26]]
这几页对本章的核心概念和公式进行了总结。

* **电容器**：是储存电能的无源元件，能量储存在电场中。
* **基本关系**：
    * $Q = CV$
    * $i(t) = C \frac{dv(t)}{dt}$
    * $v(t) = \frac{1}{C} \int i(\lambda)d\lambda$
    * $w(t) = \frac{1}{2}Cv^{2}(t)$
* **串联** (第25页)：
    * $\frac{1}{C_{eq}}=\frac{1}{C_{1}}+\frac{1}{C_{2}}+...+\frac{1}{C_{n}}$
    * 总电容减小，类似于电阻并联。
* **并联** (第25页)：
    * $C_{eq}=C_{1}+C_{2}+...+C_{n}$
    * 总电容增大，类似于电阻串联。
