# 实验 5：运算放大器 (Operational Amplifiers) - 解析版

## 1. 实验简介与基础 (Introduction)

本实验旨在掌握 LM741 运算放大器的三种基本配置：反相放大器、同相放大器以及积分器（有源滤波器）。

**核心参数：**
* **电源电压 ($V_{ss}$):** $\pm 10V$ (部分步骤需调整为 $\pm 15V$)
* **芯片模型:** LM741 (通用型运放)

### 引脚功能速查
*(基于 Figure 5-1)*

| 引脚 | 名称 | 功能描述 |
| :--- | :--- | :--- |
| 2 | **Inverting Input (-)** | 反相输入端。信号从此输入，输出相位翻转 180°。 |
| 3 | **Non-Inverting Input (+)** | 同相输入端。信号从此输入，输出相位与输入相同。 |
| 6 | **Output** | 信号输出端。电压受电源轨 ($V^+/V^-$) 限制。 |
| 7 | **$V^+$** | 正电源输入 (例如 +10V)。 |
| 4 | **$V^-$** | 负电源输入 (例如 -10V)。 |

---

## 2. 基础特性测试 (Figure 5-2 & 5-3)

### 步骤 1：开环/比较器测试 (Figure 5-2)
* **电路状态：** 输入 $V_{in}=+0.1V$ 接同相端 (+)，反相端 (-) 通常接地或用于比较。
* **原理分析：** 由于没有反馈回路（Open Loop），运放处于**开环增益**极高的状态（通常 $10^5$ 到 $10^6$）。
    公式：$V_{out} = A_{OL} \times (V_+ - V_-)$
* **解析结果：** 只要 $V_+ > V_-$ (即 $0.1V > 0V$)，输出就会瞬间达到正向饱和电压 ($+V_{sat}$)。
    > **预期 $V_{out}$：** 接近正电源电压 (约 $+9V$ 到 $+10V$)。

### 步骤 2：电压跟随器 (Voltage Follower, Figure 5-3)
* **电路状态：** 输出端直接短路连接回反相输入端 (-)。
* **原理分析：** 这是典型的**单位增益缓冲器 (Unity Gain Buffer)**。
    公式：$V_{out} = V_{in}$
* **解析结果：**
    > **理想增益：** **1** (或者 0 dB)。
    > **作用：** 阻抗匹配（高输入阻抗，低输出阻抗），隔离信号源与负载。

---

## 3. 反相放大器 (Inverting Configuration)

### 步骤 3 & 4：直流增益分析 (Figure 5-4)
**理论公式：**
$$Gain (A_v) = -\frac{R_2}{R_1}$$
$$V_{out} = V_{in} \times (-\frac{R_2}{R_1})$$

**实验参数：** $V_{in} = +0.1V$, $R_1 = 1 k\Omega$。

| $R_2$ 值 | 理论增益计算 ($-\frac{R_2}{R_1}$) | 理想输出 $V_{out}$ | 实验现象/实际 $V_{out}$ |
| :--- | :--- | :--- | :--- |
| **$2 k\Omega$** | $-2$ | $-0.2V$ | $-0.2V$ (正常线性区) |
| **$5 k\Omega$** | $-5$ | $-0.5V$ | $-0.5V$ (正常线性区) |
| **$10 k\Omega$** | $-10$ | $-1.0V$ | $-1.0V$ (正常线性区) |
| **$50 k\Omega$** | $-50$ | $-5.0V$ | $-5.0V$ (正常线性区) |
| **$100 k\Omega$** | $-100$ | $-10.0V$ | $\approx -9V$ (**开始饱和**) |
| **$2 M\Omega$** | $-2000$ | $-200V$ (理论) | $\approx -9V$ (**严重饱和**) |

### 步骤 5：饱和现象 (Saturation) 解析
> **问题：** 当 $R_2 = 2 M\Omega$ 时，为什么实际值与理论值差异巨大？
> **详细解答：**
> * **理论值：** 增益为 -2000，理论输出应为 $-2000 \times 0.1V = -200V$。
> * **实际限制：** 运算放大器是有源器件，其输出电压**不能超过电源电压**。
> * **结论：** 在 $\pm 10V$ 供电下，最大输出只能达到约 $\pm 9V$ (取决于运放的 Output Swing 性能)。这被称为**削波 (Clipping)** 或**饱和 (Saturation)**。

### 步骤 6 & 7：交流信号与电源影响
**设置：** $R_1=2k\Omega, R_2=200k\Omega$。
* **增益：** $A_v = -200/2 = -100$。
* **输入：** 正弦波，振幅 $0.1V$ (峰值)。
* **理想输出：** $0.1V \times 100 = 10V$ (峰值)。

**分析：**
1.  **供电 $\pm 10V$ 时：** 理想输出峰值是 $10V$，这正好处于或略微超过了电源轨的限制。波形的顶部和底部会被“削平”（失真）。
2.  **供电 $\pm 15V$ 时：** 电源范围变为 $+15V$ 到 $-15V$。现在 $10V$ 的输出信号完全在安全范围内。
    > **现象：** 增加电源电压后，输出波形不再失真，恢复为完美的正弦波。

## 4. 同相放大器 (Non-Inverting Configuration)

### 步骤 8：增益计算 (Figure 5-5)
**理论公式：**
$$Gain (A_v) = 1 + \frac{R_2}{R_1}$$

**实验参数：** $V_{in} = 0.1V$, $R_1 = 1 k\Omega$。

| 电阻组合 ($R_1 / R_2$) | 计算过程 | 增益结果 |
| :--- | :--- | :--- |
| **$1k\Omega$ / $1k\Omega$** | $1 + (1/1)$ | **2** |
| **$1k\Omega$ / $5k\Omega$** | $1 + (5/1)$ | **6** |
| **$1k\Omega$ / $10k\Omega$** | $1 + (10/1)$ | **11** |

> **问题：** 最小增益是多少？
> **回答：** 对于同相放大器，最小增益为 **1** (当 $R_2=0$ 或 $R_1=\infty$ 时)。在本实验配置下，最小增益为 2。

### 步骤 9：设计题解析
**目标：** 增益 $A_v = 40$，已知 $R_1 = 1 k\Omega$。
**计算：**
$$40 = 1 + \frac{R_2}{1 k\Omega}$$
$$39 = \frac{R_2}{1 k\Omega}$$
$$R_2 = 39 k\Omega$$

> **答案：** 需要选择 **$39 k\Omega$** 的电阻作为 $R_2$。

## 5. 积分器与滤波器 (Integrator/Filter)

### 步骤 10：波形分析 (Figure 5-6)

此电路为**带反馈电阻的积分器**（实际上是一个有源低通滤波器）。
* $R_1 = 10k\Omega$
* $R_2 = 100k\Omega$ (提供 DC 反馈路径，限制低频增益)
* $C_1 = 0.05\mu F$

**a) 有 $R_2$ 时的输出波形 (Leaky Integrator)**
* **输入：** 0V 到 1V 的方波。
* **分析：** 这是一个反相放大电路。
    * 当输入跳变到 +1V 时，输出试图向负方向变化。
    * 电容 $C_1$ 阻止电压突变，导致输出呈现**指数级下降**（充电曲线）。
    * 最终稳定值 (DC gain)：$V_{out} = - (R_2/R_1) \times V_{in} = -10 \times 1V = -10V$。
* **波形描述：** 输出是**反向的**，且不是直线，而是像鲨鱼鳍一样的**弯曲指数曲线**，逐渐逼近 -10V。

**b) 移除 $R_2$ 后的输出波形 (Ideal Integrator)**
* **分析：** 移除 $R_2$ 后成为理想积分器。
    * 公式：$V_{out} = -\frac{1}{R_1 C_1} \int V_{in} dt$。
    * 当输入为常数 (+1V) 时，积分为线性函数 ($kt$)。
* **波形描述：** 输出为**线性斜坡 (Linear Ramp)** 向下倾斜。
    * **注意：** 在实际实验中，如果没有复位机制，理想积分器很容易因为微小的输入失调电压而漂移至饱和状态。但在示波器的短时间观察中，应该能看到笔直的下降斜线，直到达到电源电压饱和。

---

## 实验总结 (Key Takeaways)

1.  **增益限制：** 无论计算出的理论增益多大，实际输出电压永远无法超过电源电压 ($V_{ss}$)。
2.  **相位关系：**
    * 反相放大器：输出与输入反相 (180°)。
    * 同相放大器：输出与输入同相 (0°)。
3.  **积分器行为：** 电容在反馈回路中会将方波转换为三角波（或斜坡），起到平滑突变的作用。
