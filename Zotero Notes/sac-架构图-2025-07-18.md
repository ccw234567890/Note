## 核心组件说明

### 1. Actor（策略网络）— $$\\pi_\\phi(a_t|s_t)$$
- 输入：当前状态 $$s_t$$
- 输出：动作分布（通常为对角高斯分布，均值 + 标准差由神经网络输出）
- 通过**重参数化技巧**采样动作：$$a_t = f_\\phi(\\epsilon_t; s_t), \\quad \\epsilon_t \\sim \\mathcal{N}(0, I)$$

### 2. Critic（双Q网络）— $$Q_{\\theta_1}, Q_{\\theta_2}$$
- 输入：状态 $$s_t$$ + 动作 $$a_t$$
- 输出：Soft Q-value（期望折扣累计奖励 + 熵）
- 使用 **Clipped Double Q** 技巧：取两个Q网络的最小值，减少过估计
- 目标Q网络 $$Q_{\\bar{\\theta}}$$ 通过 EMA 更新：$$\\bar{\\theta} \\leftarrow \\tau \\theta + (1-\\tau) \\bar{\\theta}$$

### 3. Soft Q 函数定义
$$Q^\\pi(s_t, a_t) = r(s_t, a_t) + \\gamma \\mathbb{E}_{s_{t+1}\\sim p}\\left[V^\\pi(s_{t+1})\\right]$$

$$V^\\pi(s_t) = \\mathbb{E}_{a_t\\sim\\pi}\\left[Q^\\pi(s_t, a_t) - \\alpha \\log \\pi(a_t|s_t)\\right]$$

### 4. 温度系数 $$\\alpha$$
- 自动调节，平衡**奖励最大化**与**熵最大化**
- 目标熵 $$\\bar{\\mathcal{H}} = -\\dim(\\mathcal{A})$$（动作空间维度的负值）
- 当策略熵低于目标 → 增大 $$\\alpha$$（鼓励探索）
- 当策略熵高于目标 → 减小 $$\\alpha$$（鼓励利用）

### 5. 训练流程

1. **采样**：从回放缓冲区采样 batch $$(s_t, a_t, r_t, s_{t+1})$$
2. **更新Q网络**：最小化 Soft Bellman 残差 $$J_Q(\\theta)$$
3. **更新策略网络**：最小化 KL 散度 $$J_\\pi(\\phi)$$
4. **更新温度**：最小化 $$J(\\alpha)$$
5. **软更新目标网络**：$$\\bar{\\theta} \\leftarrow \\tau \\theta + (1-\\tau) \\bar{\\theta}$$
