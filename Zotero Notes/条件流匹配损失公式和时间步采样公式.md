
# 条件流匹配损失公式和时间步采样公式

## 一、条件流匹配损失公式

### 1.1 完整公式

$$\mathcal{L}_\tau(\theta) = \mathbb{E}_{p(A_t|o_t), q(A_t^\tau|A_t)} \left\| v_\theta(A_t^\tau, o_t) - u(A_t^\tau|A_t) \right\|^2$$

### 1.2 每个符号的意义和作用

| 符号                         | 含义            | 作用                                                                             |                                                                                                         |     |
| -------------------------- | ------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------- | --- |
| $\mathcal{L}_\tau(\theta)$ | 条件流匹配损失函数     | 衡量模型预测的向量场与真实向量场之间的均方误差，是训练过程中需要最小化的目标函数                                       |                                                                                                         |     |
| $\theta$                   | 模型的可学习参数      | 包括 Transformer 权重、动作专家的 MLP 权重、线性投影层等所有需要梯度更新的参数                               |                                                                                                         |     |
| $\mathbb{E}$               | 期望算子          | 对括号内的随机变量分布求期望，表示对所有可能的 $(A_t, o_t)$ 对和所有可能的噪声路径取平均                            |                                                                                                         |     |
| $p(A_t\|o_t)$              |               | 给定观测 $o_t$ 下的真实动作分布                                                            | 数据分布——从真实机器人数据集中采样得到的、在观测 $o_t$ 条件下应该执行的动作块 $A_t$ 的条件分布                                                 |     |
| $q(A_t^\tau\|A_t)$         |               | 条件概率路径                                                                         | 给定干净动作 $A_t$ 的条件下，噪声动作 $A_t^\tau$ 的条件分布。定义为线性高斯路径：$q(A_t^\tau\|A_t) = \mathcal{N}(\tau A_t, (1-\tau)I)$ |     |
| $v_\theta(A_t^\tau, o_t)$  | 模型学习的向量场      | 神经网络输出的向量场，输入是噪声动作 $A_t^\tau$ 和观测 $o_t$，输出是一个与动作同维度的向量，指示去噪方向                  |                                                                                                         |     |
| $u(A_t^\tau\|A_t)$         |               | 真实条件向量场（目标向量场）                                                                 | 理论上将噪声动作 $A_t^\tau$ 推向干净动作 $A_t$ 的方向，是模型需要拟合的目标                                                         |     |
| $\|\cdot\|^2$              | 均方误差（L2 范数平方） | 损失度量方式，计算预测向量场与目标向量场之间每个分量的平方差之和                                               |                                                                                                         |     |
| $\tau$                     | 流匹配时间步        | $\tau \in [0,1]$，控制噪声与信号的混合比例。$\tau=0$ 时 $A_t^0$ 为纯噪声，$\tau=1$ 时 $A_t^1$ 为干净动作 |                                                                                                         |     |
| $A_t$                      | 干净动作块         | $A_t = [a_t, a_{t+1}, \dots, a_{t+H-1}]$，长度为 $H=50$ 的未来连续动作序列                  |                                                                                                         |     |
| $A_t^\tau$                 | 噪声动作块         | $A_t^\tau = \tau A_t + (1-\tau)\varepsilon$，干净动作与噪声的线性插值结果                     |                                                                                                         |     |
| $o_t$                      | 观测            | $o_t = [I_{t1}, \dots, I_{tn}, l_t, q_t]$，包含多张 RGB 图像、语言指令和机器人本体感知状态（关节角度）     |                                                                                                         |     |

### 1.3 公式的实践展开

在实际训练中，上述期望通过以下步骤实现：

1. **从数据集中采样**：从 $p(A_t|o_t)$ 中采样一个真实的 $(o_t, A_t)$ 对
2. **采样噪声**：$\varepsilon \sim \mathcal{N}(0, I)$，从标准正态分布中采样一个与 $A_t$ 同维度的随机噪声向量
3. **构造噪声动作**：$A_t^\tau = \tau A_t + (1-\tau)\varepsilon$，将干净动作与噪声按时间步 $\tau$ 线性混合
4. **计算目标向量场**：$u(A_t^\tau|A_t) = A_t - \varepsilon$，这是将噪声动作推回干净动作的方向
5. **模型前向传播**：$v_\theta(A_t^\tau, o_t)$，将噪声动作和观测输入模型，得到预测的向量场
6. **计算损失**：$\|v_\theta(A_t^\tau, o_t) - (A_t - \varepsilon)\|^2$，计算预测与目标的均方误差

> During training, we supervise these action tokens using a conditional flow matching loss [28, 32],
> $$\mathcal{L}_\tau(\theta) = \mathbb{E}_{p(A_t|o_t), q(A_t^\tau|A_t)} \left\| v_\theta(A_t^\tau, o_t) - u(A_t^\tau|A_t) \right\|^2,$$
> where subscripts denote robot timesteps and superscripts denote flow matching timesteps, with $\tau \in [0,1]$.

(Black 等, 2026)

### 1.4 线性高斯概率路径

条件概率路径的具体形式为：

$$q(A_t^\tau|A_t) = \mathcal{N}(\tau A_t, (1-\tau)I)$$

这表示 $A_t^\tau$ 服从以 $\tau A_t$ 为均值、$(1-\tau)I$ 为协方差矩阵的高斯分布。当 $\tau=0$ 时，$q(A_t^0|A_t) = \mathcal{N}(0, I)$ 为标准正态分布（纯噪声）；当 $\tau=1$ 时，$q(A_t^1|A_t) = \mathcal{N}(A_t, 0)$ 为狄拉克分布（完全确定性的干净动作）。

> Recent work in high-resolution image [14] and video [38] synthesis has shown that flow matching can achieve strong empirical performance when combined with a simple linear Gaussian (or optimal transport) probability path [28], given by $q(A_t^\tau|A_t) = \mathcal{N}(\tau A_t, (1-\tau)I)$.

(Black 等, 2026)

### 1.5 目标向量场的解析形式

从线性高斯路径可以解析推导出条件向量场：

$$u(A_t^\tau|A_t) = A_t - \varepsilon$$

推导过程：给定 $A_t^\tau = \tau A_t + (1-\tau)\varepsilon$，向量场定义为概率路径的漂移项。对于线性插值路径，向量场恰好等于干净动作减去噪声。

> In practice, the network is trained by sampling random noise $\varepsilon \sim \mathcal{N}(0, I)$, computing the "noisy actions" $A_t^\tau = \tau A_t + (1-\tau)\varepsilon$, and then training the network outputs $v_\theta(A_t^\tau, o_t)$ to match the denoising vector field $u(A_t^\tau|A_t) = A_t - \varepsilon$.

(Black 等, 2026)

## 二、时间步采样公式

### 2.1 完整公式

$$p(\tau) = \text{Beta}\left(\frac{s-\tau}{s}; 1.5, 1\right), \quad s = 0.999$$

### 2.2 每个符号的意义和作用

| 符号 | 含义 | 作用 |
|------|------|------|
| $p(\tau)$ | 时间步 $\tau$ 的概率密度函数 | 定义训练时每个 $\tau$ 值被采样到的概率，控制模型在不同噪声水平上的学习重点 |
| $\tau$ | 流匹配时间步 | $\tau \in [0, s]$，即采样范围被截断在 $[0, 0.999]$ 之间，$\tau \geq s$ 的时间步永远不会被采样 |
| $\text{Beta}(\cdot; \alpha, \beta)$ | Beta 分布的概率密度函数 | 定义在 $[0,1]$ 区间上的连续概率分布，形状由参数 $\alpha$ 和 $\beta$ 控制 |
| $\frac{s-\tau}{s}$ | 自变量变换 | 将 $\tau \in [0, s]$ 映射到 $[0, 1]$ 区间，使得 Beta 分布可以应用于截断后的范围 |
| $1.5$ | Beta 分布的形状参数 $\alpha$ | $\alpha > 1$ 使得概率密度向自变量较小的一侧偏移（即向 $\frac{s-\tau}{s}$ 接近 1 的一侧，对应 $\tau$ 接近 0） |
| $1$ | Beta 分布的形状参数 $\beta$ | $\beta = 1$ 使得概率密度在自变量较大的一侧（$\frac{s-\tau}{s}$ 接近 0，对应 $\tau$ 接近 $s$）呈均匀衰减 |
| $s$ | 截断阈值 | $s = 0.999$，高于此值的时间步不被采样。只要积分步长 $\delta > 1-s = 0.001$，模型就不需要学习 $\tau > s$ 的区域 |

### 2.3 公式的直观理解

$$\text{Beta}(x; 1.5, 1) \quad \text{其中} \quad x = \frac{s-\tau}{s}$$

- 当 $\tau = 0$（纯噪声）时：$x = \frac{0.999-0}{0.999} \approx 1$，Beta(1; 1.5, 1) 的概率密度最大
- 当 $\tau = 0.5$ 时：$x = \frac{0.999-0.5}{0.999} \approx 0.5$，概率密度中等
- 当 $\tau = 0.999$ 时：$x = \frac{0.999-0.999}{0.999} = 0$，概率密度为 0

因此，**低时间步（高噪声水平）被更频繁地采样**，模型在训练中更多地学习从高噪声状态去噪的能力。

### 2.4 设计动机

论文作者解释了为什么采用这种非均匀采样分布：

> We designed a timestep sampling distribution that emphasizes low timesteps (high noise levels); additionally, timesteps above a given threshold $s$ are not sampled at all, since they are not needed so long as the integration step $\delta$ is greater than $1-s$.

(Black 等, 2026)

**与传统图像生成的区别**：

在图像生成中，标准做法是从均匀分布 $\tau \sim U(0,1)$ 或对数正态分布中采样。但机器人动作预测与图像生成有本质区别：

1. **观测的约束力不同**：图像生成中，文本标签对图像的约束较弱，模型在低噪声区域只需学习恒等映射，在高噪声区域只需学习数据均值。但机器人观测 $o_t$ 对动作的约束远强于文本对图像的约束——给定观测，可能的动作分布非常集中。

2. **预测均值更难**：$\mathbb{E}[A_t|o_t]$（给定观测预测平均动作）本身就是一个困难的回归问题，不像图像生成中预测平均图像那样简单。

3. **高噪声区域更需要学习**：因此模型需要更多地在高噪声区域（低 $\tau$）进行训练，以充分学习从噪声中恢复精确动作的能力。

> However, we hypothesize that the task of action prediction is subtly different from high-resolution image synthesis — while it may be relatively easy to predict the mean image conditioned on a text label, predicting the mean action conditioned on a robot observation (i.e., learning $\mathbb{E}[A_t|o_t]$) is a much harder problem; this is because the observation $o_t$ is very informative in that it should constrain the distribution of possible actions much more than a text label constrains the distribution of possible images.

(Black 等, 2026)

### 2.5 截断阈值 $s$ 的意义

$s=0.999$ 意味着 $\tau \in [0.999, 1]$ 的区域永远不会被采样。这在实际中是完全可行的，因为推理时使用 10 步欧拉积分（$\delta=0.1$），每一步的步长远大于 $1-s = 0.001$。模型不需要学习 $\tau > 0.999$ 区域的向量场，因为积分永远不会到达那里——最后一步从 $\tau=0.9$ 直接跳到 $\tau=1.0$。

> We use $s = 0.999$ in our experiments, which allows for $\delta > \frac{1}{1000}$, or up to 1,000 integration steps.

(Black 等, 2026)

## 三、两个公式的协同关系

条件流匹配损失公式和时间步采样公式在训练中协同工作：

1. **损失公式**定义了模型在每个时间步 $\tau$ 上的学习目标——让预测向量场 $v_\theta$ 逼近真实向量场 $u$
2. **采样公式**决定了模型在哪些 $\tau$ 值上花更多精力学习——偏向高噪声区域（低 $\tau$）

两者的结合使得 π₀ 能够高效地学习从纯噪声到精确机器人动作的连续映射，同时将计算资源集中在最需要学习的噪声水平上。

## 四、电子信息工程视角

### 4.1 损失公式 → 自适应均衡器的训练准则

条件流匹配损失 $\mathcal{L}_\tau(\theta) = \mathbb{E}\|v_\theta - u\|^2$ 类似于通信系统中**自适应均衡器**的**最小均方误差（MMSE）准则**：

- $A_t$（干净动作）→ 发送端的原始信号
- $\varepsilon$（高斯噪声）→ 信道加性高斯白噪声（AWGN）
- $A_t^\tau = \tau A_t + (1-\tau)\varepsilon$ → 接收端收到的带噪信号，信噪比由 $\tau$ 控制
- $u(A_t^\tau|A_t) = A_t - \varepsilon$ → 理想均衡器的输出（恢复的信号减去噪声）
- $v_\theta(A_t^\tau, o_t)$ → 自适应均衡器的实际输出
- $\|v_\theta - u\|^2$ → 均衡器输出与理想输出之间的瞬时误差

### 4.2 采样公式 → 非均匀训练信噪比分布

时间步采样 $p(\tau) = \text{Beta}((s-\tau)/s; 1.5, 1)$ 相当于在训练时**优先使用低信噪比的样本**进行训练：

- $\tau$ 小 → 信号分量 $\tau A_t$ 弱，噪声分量 $(1-\tau)\varepsilon$ 强 → 低信噪比（SNR）
- $\tau$ 大 → 信号分量强，噪声分量弱 → 高信噪比

在通信系统中，均衡器通常在低 SNR 条件下更难收敛，因此需要更多低 SNR 的训练样本。这与 π₀ 的设计思路完全一致——模型需要更多地在高噪声条件下学习，才能在实际推理时从纯噪声中准确恢复出动作信号。
