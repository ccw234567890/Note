Space Mouse 是一种**3D空间鼠标**，在SERL论文中用于**遥操作采集演示数据**。

---

### 什么是Space Mouse？

Space Mouse（也叫3Dconnexion SpaceMouse）是一种**六自由度输入设备**，外形像一个带摇杆的小圆盘：

```
         ┌──────────┐
         │  圆盘/帽  │  ← 可以推、拉、扭、倾斜
         │  ●━━━━━● │
         │  ┃  ○  ┃ │
         │  ●━━━━━● │
         └──────────┘
              ↑
         底座（固定）
```

### 工作原理

与普通鼠标不同，Space Mouse**不测量位移**，而是**测量施加在圆盘上的力**：

|操作|效果|
|---|---|
|**向前推**|末端执行器沿X轴平移|
|**向后拉**|末端执行器沿X轴反向平移|
|**向左推**|沿Y轴平移|
|**向上提**|沿Z轴平移|
|**扭转**|绕Z轴旋转（偏航）|
|**倾斜**|绕X/Y轴旋转（俯仰/翻滚）|

每个方向对应一个自由度，总共**6个自由度**（3个平移 + 3个旋转）。

### 在SERL中的用途

> We collect 20 teleoperated demonstrations using a Space Mouse for each task.

(Luo 等, 2025)

具体流程：

1. 操作员用手**推/拉/扭**Space Mouse的圆盘
2. Space Mouse输出6维力/扭矩信号
3. 这些信号被映射为**末端执行器的6D增量位姿**
4. 机器人实时跟随操作员的意图运动
5. 操作员通过腕部相机画面观察机器人视角
6. 完成一次任务操作 → 记录为一条**演示轨迹**

### 为什么用Space Mouse而不是游戏手柄或VR？

|设备|自由度|特点|
|---|---|---|
|**Space Mouse**|6DoF|单手同时控制平移+旋转，直觉操作|
|游戏手柄|4DoF（两个摇杆各2DoF）|需要切换模式才能控制旋转|
|VR手柄|6DoF|精度不够，有延迟|
|数据手套|高自由度|昂贵，设置复杂|

**Space Mouse的核心优势**：单手就能同时控制6个自由度，操作员可以**平滑连续地**控制机器人末端运动，非常适合采集高质量的演示数据。

### 类比

```
普通鼠标：在2D平面上移动光标（2自由度）
Space Mouse：在3D空间中移动机器人末端（6自由度）

普通鼠标：你移动它，它测量位移
Space Mouse：你推它，它测量力
```
