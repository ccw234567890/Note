# TFCDiff: Robust ECG Denoising via Time-Frequency Complementary Diffusion

## 任务性质：条件生成（Conditional Generation），不是分类

### 核心结论

TFCDiff 做的任务是**条件生成（conditional generation）**，本质是**回归任务**——从带噪信号 $\tilde{x}$ 出发，逐步去噪生成干净信号 $x_0$，输出是 3600 个连续电压值，而不是离散类别标签。

### 论文原文定义

> "Diffusion models have emerged as a powerful class of **deep generative models**."
> "Further developments by [25] extended diffusion models to **conditional generation**."

(Li et al., 2025)

> "In the reverse process p, we iteratively **recover the clean signal** by denoising $x_T$ using a parameterized model $p(x_{t-1}|x_t, \tilde{x})$, where the observed noisy signal $\tilde{x}$ serves as a **conditioning input**."

(Li et al., 2025)

### 分类 vs 回归/生成 对比

| 维度 | 分类 (Classification) | 回归/生成 (Regression/Generation) |
|:---|:---|:---|
| **输出** | 离散类别标签（如"正常/异常"） | 连续值信号（如 3600 个采样点） |
| **损失函数** | 交叉熵 (Cross-Entropy) | L1/L2 距离 |
| **论文的损失** | — | $\mathcal{L}(\theta) = \mathbb{E}[\|\varepsilon - \varepsilon_\theta\|_1]$ |
| **评估指标** | 准确率、F1 分数 | SSD、MAD、PRD、CosSim、ImSNR |
| **任务本质** | 判断"是什么" | 预测"是什么数值" |

### 数据流

```
输入：带噪信号 x̃（连续值，3600 个采样点）
        │
        ▼
条件扩散模型（TFCDiff）
        │
        ▼
输出：干净信号 x₀（连续值，3600 个采样点）
```

这是一个**逐点回归任务**——模型需要为每个时间点预测一个精确的数值（电压值），而不是判断它属于哪个类别。

### 为什么容易混淆为"分类"？

扩散模型在**每一步**做一个"预测噪声"的操作：

```
x_t → U-Net → 预测 ε̂
```

但这本质上是**回归**：预测每个位置的噪声数值，而不是分类标签。

### 直观类比

| 任务 | 类比 |
|:---|:---|
| **分类** | 医生看 ECG 说"这是房颤" |
| **回归/生成** | 画家根据模糊照片还原清晰照片——每个像素的颜色值都要精确预测 |

TFCDiff 做的是后者：给定一张"模糊的 ECG"，还原出"清晰的 ECG"，每个时间点的电压值都要精确匹配。

### 训练损失

$$\mathcal{L}(\theta) = \mathbb{E}_{x_0, \tilde{x}, t, \varepsilon} \left[ \| \varepsilon - \varepsilon_\theta(x_t, t, \tilde{x}) \|_1 \right]$$

- $\varepsilon$：真实噪声（已知）
- $\varepsilon_\theta(x_t, t, \tilde{x})$：模型预测的噪声
- L1 损失：预测噪声与真实噪声的绝对误差

### 评估指标

五个定量指标，全部基于**连续数值**的误差度量：

- **SSD (Sum of Squared Distances)**：平方误差和，越低越好
- **MAD (Maximum Absolute Distance)**：最大绝对误差，越低越好
- **PRD (Percentage Root-Mean-Square Difference)**：百分比均方根误差，越低越好
- **CosSim (Cosine Similarity)**：余弦相似度，越高越好
- **ImSNR (Improved SNR)**：信噪比提升，越高越好

没有使用准确率、F1 分数等分类指标。

## 一句话总结

> **TFCDiff 做的是条件生成（conditional generation），本质是回归任务——从带噪信号 $\tilde{x}$ 出发，逐步去噪生成干净信号 $x_0$，输出是 3600 个连续电压值，而不是离散类别标签。**
