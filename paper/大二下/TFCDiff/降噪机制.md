# TFCDiff: Robust ECG Denoising via Time-Frequency Complementary Diffusion

## 时域与频域的相互变换及去噪过程

### 一、时域 ↔ 频域：DCT 与 IDCT

#### 1.1 什么是时域和频域？

**时域（Time Domain）：** 信号以时间为横轴，电压为纵轴。ECG 时域信号表示为：

$$L = [l_0, l_1, l_2, ..., l_{3599}]$$

每个 $l_n$ 是第 $n$ 个采样点的电压值。

**频域（Frequency Domain）：** 信号以频率为横轴，幅度为纵轴，表示各频率成分的强度：

$$D = [d_0, d_1, d_2, ..., d_{3599}]$$

每个 $d_k$ 是第 $k$ 个频率分量的幅度。

#### 1.2 DCT 正变换：时域 → 频域（公式 1）

$$D(k) = c(k) \sum_{n=0}^{N-1} L(n) \cos\left(\frac{(2n+1)k\pi}{2N}\right) \tag{1}$$

**直观理解：** 把时域信号 $L$ 与不同频率的余弦波做内积（点乘求和）。若信号在某频率上有成分，内积结果大；否则接近零。

- $k=0$：余弦波是常数 1，$D(0)$ 即信号均值（DC 分量）
- $k=1$：余弦波为一个完整周期，$D(1)$ 是基频分量
- $k$ 越大，余弦波振荡越快，$D(k)$ 对应越高频的成分

#### 1.3 IDCT 逆变换：频域 → 时域（公式 2）

$$L(n) = \sum_{k=0}^{N-1} c(k) D(k) \cos\left(\frac{(2n+1)k\pi}{2N}\right) \tag{2}$$

**直观理解：** 将每个频率分量 $D(k)$ 乘以对应的余弦波，再全部加起来，重建原始时域信号。

#### 1.4 归一化系数

$$c(k) = \begin{cases} \sqrt{1/N}, & k = 0 \\ \sqrt{2/N}, & k \neq 0 \end{cases}$$

保证变换是**正交的**——DCT 后再 IDCT 能完美还原原始信号。

(Li 等, 2025)

### 二、关键操作：截断（Truncation）

#### 2.1 为什么要截断？

ECG 的诊断信息主要分布在 **0.5–50 Hz**。高于 50 Hz 的频率成分几乎不包含有用信息，论文验证 DCT 系数的高频部分**接近零**。

#### 2.2 截断计算

采样率 $f_s = 360$ Hz，信号长度 $N = 3600$（10 秒）：

$$\Delta f = \frac{f_s}{2N} = \frac{360}{2 \times 3600} = 0.05 \text{ Hz}$$

第 $k$ 个系数对应的频率：$f_k = k \times 0.05$ Hz

要保留 50 Hz 以下成分：

$$\left[\frac{50}{0.05}\right] = 1000$$

即只保留前 **1000 个 DCT 系数**，其余截断为零。

#### 2.3 截断保真度验证

论文对 37590 个 10 秒 ECG 片段进行验证，重建信号与原始信号的 SSD 仅为 $(1.66 \pm 3.30) \times 10^{-3}$，几乎无损。

(Li 等, 2025)

### 三、去噪的核心机制

#### 3.1 整体流程

```
带噪 ECG (时域) ──DCT──→ 频域 ──截断──→ 保留前1000个系数 ──缩放──→ 归一化频域信号
                                                                      │
                                                         扩散模型在频域中迭代去噪
                                                                      │
去噪 ECG (时域) ←──IDCT── 零填充回3600 ←──反缩放 ←── 去噪后的频域系数
```

#### 3.2 为什么在频域去噪？

**核心原因：信号和噪声在频域中是可分离的。**

| 成分 | 频域位置 | 特点 |
|------|---------|------|
| ECG 诊断信息 | 0.5–50 Hz | 能量集中在前 1000 个系数 |
| 基线漂移 (BW) | 低频 (<0.5 Hz) | 集中在 DC 附近 |
| 肌肉伪迹 (MA) | 高频 | 分布在高频系数 |
| 电极运动伪迹 (EM) | 宽频 | 散布在各频段 |

**截断本身就是一种去噪：** 直接丢弃 50 Hz 以上的高频系数，去除部分高频噪声。

#### 3.3 扩散模型在频域中的去噪（公式 3-8）

在频域中，扩散模型学习从带噪 DCT 系数 $\tilde{x}$ 恢复干净 DCT 系数 $x_0$：

**前向过程（训练时）：** 逐步向干净系数 $x_0$ 添加高斯噪声：

$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \varepsilon \tag{3}$$

**反向过程（去噪时）：** 从纯噪声 $x_T$ 开始，逐步去除噪声，以带噪观测 $\tilde{x}$ 为条件：

$$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{1 - \alpha_t}{\sqrt{1 - \bar{\alpha}_t}} \varepsilon_\theta(x_t, t, \tilde{x}) \right) + \sigma_t z \tag{6}$$

**关键点：** 整个扩散过程在 DCT 频域中进行。模型 $\varepsilon_\theta$ 预测频域中的噪声，然后从频域信号中减去它。

#### 3.4 TFEM：时域特征增强（公式 14）

纯频域去噪会丢失时域细节（如 P 波、T 波的精确形态）。TFEM 模块作为"桥梁"：

**TFE（时域特征提取）：**
1. 频域特征图 → **零填充**恢复原始长度 → **1D IDCT** 转时域
2. 时域残差块处理（捕捉波形细节）
3. **1D DCT** 转回频域 → **截断**

**TFF（时域特征融合）：**
在编码器中间层，通过跨域注意力融合时域和频域表征：

$$F_{fused} = T_V \times \text{Softmax}\left( \frac{T_Q \times T_K^T}{\sqrt{C}} \right) \tag{14}$$

其中 $T_Q, T_K, T_V$ 是通过 IDCT 转换到时域的 Q、K、V 张量。

(Li 等, 2025)

### 五、总结

| 步骤 | 域 | 做了什么 | 效果 |
|------|-----|---------|------|
| **DCT** | 时域→频域 | 将 ECG 信号分解为频率分量 | 分离信号和噪声 |
| **截断** | 频域 | 只保留前 1000 个系数 | 去高频噪声，降计算量 |
| **扩散去噪** | 频域 | 迭代去除频域中的噪声 | 恢复干净频域系数 |
| **TFEM** | 时域↔频域 | 交替转换，捕捉时域细节 | 修复 P-QRS-T 波形形态 |
| **IDCT** | 频域→时域 | 重建时域信号 | 得到最终去噪 ECG |

**一句话总结：** 先把带噪 ECG 通过 DCT 转到频域，在频域中利用扩散模型迭代去噪（同时通过 TFEM 模块反复回到时域修正波形细节），最后通过 IDCT 转回时域得到干净 ECG。
