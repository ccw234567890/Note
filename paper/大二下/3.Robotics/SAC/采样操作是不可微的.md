### "采样操作不可微"是什么意思？

#### 用一个简单例子理解

假设你有一个神经网络，输出两个值：**均值 $\mu$** 和 **标准差 $\sigma$**，表示一个高斯分布 $\mathcal{N}(\mu, \sigma^2)$。

现在你要从这个分布中**采样**出一个动作 $a$：

$$a \sim \mathcal{N}(\mu, \sigma^2)$$

这个采样操作在计算图中是这样的：

```
输入 s → 神经网络 → [μ, σ] → 采样器 → a → 计算损失 L(a) → 反向传播求梯度
                              ↑
                         随机性来源
```

**问题出在采样器这里**——采样是一个**离散的随机选择**，它内部做的事情是：

1. 生成一个随机数（比如从均匀分布中抽一个值）
2. 用这个随机数"挑选"一个输出值

这个过程的计算图是**断开的**——你无法对"随机数生成"求导。

#### 具体来说

假设你采样得到 $a = 0.5$，损失 $L = (a - 1)^2 = 0.25$。

你想知道：**"如果 $\mu$ 增大一点点，损失会怎么变？"** 即求 $\frac{\partial L}{\partial \mu}$。

但问题是：**采样结果 $a=0.5$ 是随机产生的**。即使 $\mu$ 从 0 变成 0.1，下一次采样可能得到 $a=0.3$ 或 $a=0.7$，你无法确定 $a$ 随 $\mu$ 的变化关系。因为 $a$ 和 $\mu$ 之间隔了一个**不可微的随机采样节点**。

#### 用代码对比更清楚

**❌ 不可微的方式（直接采样）：**

```
mu, sigma = policy_network(state)       # 输出均值和标准差
a = torch.normal(mu, sigma)             # 采样！这里断了梯度的路
loss = (a - target) ** 2
loss.backward()                         # 梯度无法传到 mu 和 sigma
```

**✅ 重参数化的方式（可微）：**

```
mu, sigma = policy_network(state)       # 输出均值和标准差
epsilon = torch.randn_like(sigma)       # 从标准高斯采样噪声（固定分布）
a = mu + sigma * epsilon                # 可微的线性变换！
loss = (a - target) ** 2
loss.backward()                         # 梯度顺利传到 mu 和 sigma
```

#### 关键区别

||直接采样|重参数化采样|
|---|---|---|
|随机性来源|采样操作本身|外部输入的固定噪声 $epsilon$|
|计算图|采样节点断开梯度|全是可微的加减乘除|
|梯度能否传播|❌ 不能|✅ 能|
|公式|$a sim mathcal{N}(mu, sigma^2)$|$a = mu + sigma cdot epsilon, epsilon sim mathcal{N}(0,1)$|

#### 回到 SAC 的语境

SAC 的策略网络输出 $\mu_\phi(s_t)$ 和 $\sigma_\phi(s_t)$，如果直接采样：

$$a_t \sim \mathcal{N}(\mu_\phi(s_t), \sigma_\phi(s_t)^2)$$

那么策略参数 $\phi$ 的梯度 $\nabla_\phi J$ 就无法计算——因为 $a_t$ 和 $\phi$ 之间隔着不可微的采样操作。

而使用重参数化技巧后：

$$a_t = \mu_\phi(s_t) + \sigma_\phi(s_t) \odot \epsilon_t, \quad \epsilon_t \sim \mathcal{N}(0, I)$$

这里 $\epsilon_t$ 是从**固定分布**（标准高斯）中采样的，与 $\phi$ **无关**。整个表达式 $a_t = \mu_\phi(s_t) + \sigma_\phi(s_t) \cdot \epsilon_t$ 全是可微的加减乘除运算，梯度可以畅通无阻地从损失函数一直传到 $\phi$。

这就是论文中公式 (8) 和 (9) 存在的根本原因：

> $$a_t = f_\phi(\epsilon_t; s_t)$$
> 
> $$J_\pi(\phi) = \mathbb{E}_{s_t\sim\mathcal{D},\epsilon_t\sim\mathcal{N}}[\alpha\log\pi_\phi(f_\phi(\epsilon_t;s_t)|s_t) - Q_\theta(s_t,f_\phi(\epsilon_t;s_t))]$$

(Haarnoja 等, 2019)

把随机性从"模型参数内部"剥离到"外部输入噪声"，让梯度能够顺利传播。
