# TFCDiff: Robust ECG Denoising via Time-Frequency Complementary Diffusion

## 偶函数的傅里叶级数只有余弦项

### 一、傅里叶级数的一般形式

对于周期为 $T$ 的函数 $f(t)$，傅里叶级数展开为：

$$f(t) = a_0 + \sum_{k=1}^{\infty} [a_k \cos(k\omega t) + b_k \sin(k\omega t)]$$

其中 $\omega = 2\pi/T$，系数由以下积分计算：

$$a_0 = \frac{1}{T} \int_{-T/2}^{T/2} f(t) dt$$

$$a_k = \frac{2}{T} \int_{-T/2}^{T/2} f(t) \cos(k\omega t) dt$$

$$b_k = \frac{2}{T} \int_{-T/2}^{T/2} f(t) \sin(k\omega t) dt$$

一般函数既有 $a_k$（余弦项系数），也有 $b_k$（正弦项系数）。

### 二、什么是偶函数？

**定义：** 满足 $f(-t) = f(t)$ 的函数称为偶函数。

**几何意义：** 关于 y 轴对称。

偶函数示例：$f(t) = t^2$，$f(t) = \cos(t)$，$f(t) = |t|$。

**偶延拓后的信号就是偶函数：**

```
原始信号:  x(0), x(1), x(2), ..., x(N-1)
                    ↓
偶延拓后:  x(0), x(1), ..., x(N-1), x(N-1), ..., x(1)
```

验证：以中心为对称轴，左右镜像对称。

### 三、为什么偶函数的正弦项系数 $b_k = 0$？

#### 3.1 关键观察

正弦函数 $\sin(k\omega t)$ 是**奇函数**：

$$\sin(-k\omega t) = -\sin(k\omega t)$$

#### 3.2 奇函数 × 偶函数 = 奇函数

如果 $f(t)$ 是偶函数，那么 $f(t) \cdot \sin(k\omega t)$：
- $f(t)$：偶函数
- $\sin(k\omega t)$：奇函数
- **乘积：奇函数**（偶 × 奇 = 奇）

#### 3.3 奇函数在对称区间上的积分为零

$$b_k = \frac{2}{T} \int_{-T/2}^{T/2} f(t) \sin(k\omega t) dt = 0$$

**直观理解：** 奇函数在 $[-T/2, 0]$ 和 $[0, T/2]$ 上的面积大小相等、符号相反，互相抵消。

#### 3.4 余弦项系数 $a_k$ 不受影响

余弦函数 $\cos(k\omega t)$ 是**偶函数**：

$$\cos(-k\omega t) = \cos(k\omega t)$$

偶函数 × 偶函数 = 偶函数，在对称区间上的积分不为零：

$$a_k = \frac{2}{T} \int_{-T/2}^{T/2} f(t) \cos(k\omega t) dt \neq 0$$

### 四、数学证明

设 $f(t)$ 是偶函数，即 $f(-t) = f(t)$。

**正弦项系数：**

$$b_k = \frac{2}{T} \int_{-T/2}^{T/2} f(t) \sin(k\omega t) dt$$

拆成两半：

$$b_k = \frac{2}{T} \left[ \int_{-T/2}^{0} f(t) \sin(k\omega t) dt + \int_{0}^{T/2} f(t) \sin(k\omega t) dt \right]$$

对第一项做变量替换 $u = -t$：

$$\int_{-T/2}^{0} f(t) \sin(k\omega t) dt = \int_{T/2}^{0} f(-u) \sin(-k\omega u) (-du)$$

由于 $f(-u) = f(u)$（偶函数），$\sin(-k\omega u) = -\sin(k\omega u)$：

$$= \int_{T/2}^{0} f(u) [-\sin(k\omega u)] (-du) = \int_{T/2}^{0} f(u) \sin(k\omega u) du$$

$$= -\int_{0}^{T/2} f(u) \sin(k\omega u) du$$

代入原式：

$$b_k = \frac{2}{T} \left[ -\int_{0}^{T/2} f(u) \sin(k\omega u) du + \int_{0}^{T/2} f(t) \sin(k\omega t) dt \right] = 0$$

**证毕。**

### 五、回到 DCT 的上下文

#### 5.1 DCT 的偶延拓

DCT 对原始信号做偶延拓：

```
原始信号:  x(0), x(1), ..., x(N-1)
                    ↓
偶延拓后:  x(0), x(1), ..., x(N-1), x(N-1), ..., x(1)
                    ↑                          ↑
                原始部分                    镜像对称部分
```

这个延拓后的信号是**偶函数**（以 $N-0.5$ 为对称轴）。

#### 5.2 延拓后做 DFT

对偶延拓后的信号做 DFT（离散傅里叶变换），由于信号是偶函数：
- **正弦项全部消失**（$b_k = 0$）
- **只剩余弦项**（$a_k \neq 0$）
- 所有系数都是**实数**

#### 5.3 化简得到 DCT 公式

经过代数化简，就得到了 Type-II DCT 公式：

$$D(k) = c(k) \sum_{n=0}^{N-1} L(n) \cos\left(\frac{(2n+1)k\pi}{2N}\right) \tag{1}$$

**注意：** 公式中只有 $\cos$，没有 $\sin$。

### 六、总结

| 概念 | 说明 |
|------|------|
| **偶函数** | $f(-t) = f(t)$，关于 y 轴对称 |
| **奇函数** | $f(-t) = -f(t)$，关于原点对称 |
| **$\sin$ 是奇函数** | $\sin(-x) = -\sin(x)$ |
| **$\cos$ 是偶函数** | $\cos(-x) = \cos(x)$ |
| **偶 × 奇 = 奇** | 偶函数乘以奇函数是奇函数 |
| **奇函数对称积分为零** | $\int_{-a}^{a} \text{奇函数} = 0$ |
| **结论** | 偶函数的傅里叶级数中，正弦项系数全为零，只剩余弦项 |

**一句话：** 因为 DCT 先把信号延拓成偶函数，而偶函数与正弦函数（奇函数）的乘积在对称区间上积分等于零，所以正弦项全部消失，只剩下余弦项——这就是 DCT 名字中"余弦"二字的由来。
