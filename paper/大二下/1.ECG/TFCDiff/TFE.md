# TFCDiff: Robust ECG Denoising via Time-Frequency Complementary Diffusion

## TFE（时域特征提取）逐步骤详解

论文原文描述：

> "In TFE, to convert a feature map from the DCT domain to the time domain, zero padding is first applied to restore its original length, followed by 1D IDCT. After processing through a residual block, the feature map is transformed back into the frequency domain by 1D DCT and subsequent truncation."

(Li et al., 2025)

### 第一步：频域特征图 → 零填充恢复原始长度 → 1D IDCT 转时域

#### 为什么要零填充？

在 DCT 域中，特征图是**截断后**的——只保留了前 $K$ 个低频系数（对应 50 Hz 以下成分），丢弃了高频部分。

```
原始 DCT 系数（长度 N=3600）：
┌──────────────────────────────────────────────────┐
│ DC │  AC₁  │  AC₂  │ ... │  AC₉₉₉  │ AC₁₀₀₀~AC₃₅₉₉ │
└──────────────────────────────────────────────────┘
  保留前 1000 个系数（≤50Hz）        丢弃（>50Hz）

截断后的特征图（长度 K=1000）：
┌──────────────────────────────┐
│ DC │  AC₁  │  AC₂  │ ... │  AC₉₉₉  │
└──────────────────────────────┘
```

**零填充**就是在截断后的系数后面补上 $N-K$ 个零，恢复到原始长度 $N$：

```
零填充后（长度 N=3600）：
┌──────────────────────────────────────────────────┐
│ DC │  AC₁  │ ... │  AC₉₉₉  │  0  │  0  │ ... │  0  │
└──────────────────────────────────────────────────┘
                         ↑ 补零（高频部分视为零）
```

**为什么需要零填充？** 因为 1D IDCT 要求输入长度等于原始信号长度 $N$，否则逆变换会得到不同长度的时域信号，无法与原始信号对齐。

#### 1D IDCT 的作用

将频域系数 $D(k)$ 变换回时域信号 $L(n)$：

$$L(n) = \sum_{k=0}^{N-1} c(k) D(k) \cos\left(\frac{(2n+1)k\pi}{2N}\right)$$

其中 $c(k) = \begin{cases} \sqrt{1/N}, & k=0 \\ \sqrt{2/N}, & k \neq 0 \end{cases}$

**直观理解：** IDCT 把"每个频率成分的强度"重新合成为"随时间变化的波形"。零填充的高频部分贡献为零，所以重建的时域信号只包含低频成分，但长度与原始信号一致。

### 第二步：时域残差块处理（捕捉波形细节）

#### 残差块的结构

```
输入（时域信号）
    │
    ▼
┌─────────────────────┐
│   Conv1D + BN + ReLU │
└─────────────────────┘
    │
    ▼
┌─────────────────────┐
│   Conv1D + BN       │
└─────────────────────┘
    │
    ▼
    ⊕ ←──── 跳跃连接（skip connection）
    │
    ▼
    ReLU
    │
    ▼
输出
```

#### 为什么在时域做残差块？

DCT 域中，每个系数代表**全局频率信息**——一个系数对应整个 10 秒信号的某个频率成分。而 ECG 的 P 波、QRS 波群、T 波是**局部时域波形**，在频域中难以精细调整。

**残差块在时域中的作用：**
- 学习**局部波形细节**的修正（比如让 R 波更尖锐、T 波更平滑）
- 残差连接保证梯度流通，避免深层网络退化

#### 类比

```
DCT 域：像"调音台"——调整每个频率的音量
  ↓ IDCT
时域：像"波形编辑器"——直接修改波形的形状
  ↓ 残差块处理
精细调整 P-QRS-T 的形态细节
```

### 第三步：1D DCT 转回频域 → 截断

#### 1D DCT 的作用

将处理后的时域信号 $L(n)$ 重新变换回频域：

$$D(k) = c(k) \sum_{n=0}^{N-1} L(n) \cos\left(\frac{(2n+1)k\pi}{2N}\right)$$

#### 为什么要截断？

DCT 变换后得到的是**完整长度 $N$ 的频域系数**，但 U-Net 的主干网络工作在**截断后的频域**（长度 $K$）。所以需要再次截断，只保留前 $K$ 个低频系数：

```
DCT 后（长度 N=3600）：
┌──────────────────────────────────────────────────┐
│ DC │  AC₁  │  AC₂  │ ... │  AC₉₉₉  │ AC₁₀₀₀~AC₃₅₉₉ │
└──────────────────────────────────────────────────┘

截断后（长度 K=1000）：
┌──────────────────────────────┐
│ DC │  AC₁  │  AC₂  │ ... │  AC₉₉₉  │
└──────────────────────────────┘
  保留（≤50Hz）                丢弃（>50Hz）
```

**为什么可以截断？** 论文已验证，截断后重建信号与原始信号的 SSD 仅为 $(1.66 \pm 3.30) \times 10^{-3}$，几乎无损。因为 ECG 的诊断信息主要分布在 0.5–50 Hz，高于 50 Hz 的成分主要是噪声。

(Li et al., 2025)

### TFE 完整流程图

```
┌─────────────────────────────────────────────────────────┐
│                    TFE 模块                              │
│                                                          │
│  频域特征图（截断后，长度 K）                              │
│         │                                                │
│         ▼ 零填充                                         │
│  频域特征图（零填充后，长度 N）                            │
│         │                                                │
│         ▼ 1D IDCT                                        │
│  时域特征图（长度 N）                                     │
│         │                                                │
│         ▼ 残差块                                         │
│  ┌─────────────────────┐                                 │
│  │ Conv1D → BN → ReLU  │  ← 学习局部波形细节             │
│  │ Conv1D → BN         │                                 │
│  │    ⊕ ← 跳跃连接     │                                 │
│  └─────────────────────┘                                 │
│         │                                                │
│         ▼ 1D DCT                                         │
│  频域特征图（完整长度 N）                                  │
│         │                                                │
│         ▼ 截断                                           │
│  频域特征图（截断后，长度 K） → 送回 U-Net 主干            │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### 各步骤总结

| 步骤 | 操作 | 目的 |
|:---:|:---|:---|
| **① 零填充** | 截断系数后补零到原始长度 | 满足 IDCT 的输入长度要求 |
| **② 1D IDCT** | 频域 → 时域 | 从全局频率表示切换到局部波形表示 |
| **③ 残差块** | Conv1D + 跳跃连接 | 精细调整 P-QRS-T 波形的局部形态细节 |
| **④ 1D DCT** | 时域 → 频域 | 将修正后的波形信息编码回频率表示 |
| **⑤ 截断** | 只保留前 K 个低频系数 | 丢弃无关高频，与 U-Net 主干对齐 |

**核心思想：** TFE 通过"频域→时域→频域"的迂回路径，让网络既能利用 DCT 域的高效计算，又能直接在时域中精细调整 ECG 波形的局部细节，实现**时频互补**。

### 补充：1D DCT 与 P-QRS-T 波形的本质区别

#### 核心概念：全局 vs 局部

**1D DCT 系数是全局的**

DCT 的每个系数 $D(k)$ 是通过**整个 10 秒信号**的加权求和计算出来的。求和范围是 $n=0$ 到 $N-1$（整个信号长度）。这意味着：
- $D(0)$（DC 系数）= 整个 10 秒信号的平均值
- $D(1)$ = 整个 10 秒信号与最低频余弦波的匹配程度
- $D(k)$ = 整个 10 秒信号与第 $k$ 个频率余弦波的匹配程度

**每个 DCT 系数都包含整个信号的信息，不区分"这是 P 波的部分"或"这是 QRS 的部分"。**

**P-QRS-T 波形是局部的**

P 波、QRS 波群、T 波在时域中占据特定的时间位置和持续时间，可以独立调整。而 DCT 系数是全局的，每个系数影响整个 10 秒信号，无法单独调整某个波形的形态。

#### 直观类比

| 表示方式 | 类比 | 对局部波形的控制力 |
|:---|:---|:---:|
| **DCT 域** | 调音台的推子——每个推子影响整首歌 | ❌ 弱——调整一个系数影响所有位置 |
| **时域** | 波形编辑器的选区——可以框选某个 R 波单独调整 | ✅ 强——调整一个点只影响该点 |

#### 为什么 TFE 要"频域→时域→频域"？

在 DCT 域中难以精细调整波形。假设一个 ECG 信号的 R 波幅度偏低，需要调高。在 DCT 域中调高某个频率系数，R 波确实变高了，但 P 波、T 波、基线也被改变了——因为每个 DCT 系数影响整个信号。

TFE 的解决方案：先在时域中用残差块精确调整每个波形的形态（"这个 R 波幅度偏低，调高 20%"），再通过 DCT 编码回频域。残差块在时域中学习的是**位置相关的局部修正量**，在时域中很容易表达，但在 DCT 域中很难表达。
