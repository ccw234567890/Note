
# π₀.₇ 论文详解 — 实验设计、结果与结论

## 一、论文概述

**π₀.₇** 由 **Physical Intelligence** 团队开发，是新一代机器人基础模型，约 **5B 参数**的视觉-语言-动作（VLA）模型，能在**零样本**条件下执行高度灵巧的操作任务，展现出跨本体迁移、指令理解、组合泛化等涌现能力。

## 二、实验设计

### 2.1 模型架构

| 组件 | 参数规模 | 说明 |
|------|---------|------|
| **VLM 骨干网络** | 4B | 基于 Gemma3 4B 初始化（含 400M 视觉编码器） |
| **动作专家（Action Expert）** | 860M | 轻量级 Transformer，使用流匹配（Flow Matching）目标 |
| **总参数量** | ~5B | — |

**架构关键设计：**
- **视觉编码器**：MEM 风格的视频历史编码器，对历史观测帧进行时空压缩
- **动作专家**：使用**流匹配（Flow Matching）** 目标预测连续动作
- **知识隔离（Knowledge Insulation, KI）训练**：VLM 骨干通过 FAST token 的离散交叉熵损失训练，动作专家梯度不回流
- **动作分块**：固定输出 50 步动作块，使用实时动作分块（RTC）处理推理延迟
- **输入**：最多 4 个摄像头视角，每视角最多 6 帧历史，最多 3 张子目标图像

### 2.2 多样化提示（Diverse Prompting）策略

训练时上下文 $C_t$ 包含：

**(a) 子任务指令（Subtask Instructions）**：整体任务描述 + 中间语义子任务，推理时支持**语言教练（Language Coaching）**

**(b) 子目标图像（Subgoal Images）**：多视角子目标图像，由轻量级**世界模型**（BAGEL 14B）实时生成

**(c) 片段元数据（Episode Metadata）**：速度（离散化为 500 步区间）、质量（1-5 分）、错误标记，使模型能从**混合质量数据**中学习

**(d) 控制模式（Control Mode）**：同时训练关节级和末端执行器两种控制模式

**训练时随机丢弃策略**：每个提示组件以一定概率被丢弃，使模型推理时能灵活使用任意子集。

### 2.3 训练数据集

| 数据来源 | 说明 |
|---------|------|
| 演示数据 | 多种机器人平台上的广泛任务演示 |
| 自主数据 | 大量策略评估数据（含失败和次优轨迹） |
| 人类干预数据 | 策略 rollout 中的人类干预记录 |
| 开源机器人数据集 | 公开的跨本体机器人数据 |
| 自我中心人类视频 | 人类第一人称操作视频 |
| 网络非机器人数据 | 目标定位、属性预测、VQA、文本预测、视频描述等 |

### 2.4 机器人平台

| 平台 | 配置 | 用途 |
|------|------|------|
| 移动双臂操作器 | 2×6 DoF 臂 + 夹爪 + 升降 + 全向底座 | 主要训练/评估 |
| 静态双臂机器人（BiPi） | 2×6 DoF 轻量臂 + 夹爪 | 主要训练/评估 |
| 双臂 UR5e | 2×6 DoF 臂 + Robotiq 夹爪 | 跨本体迁移目标 |
| 单臂 6 DoF 系统 | 6 DoF 臂 + 夹爪 | 语言泛化实验 |

### 2.5 推理流程

1. 初始化观测 $o_0$、任务指令 $l$、元数据 $m$、控制模式 $c$
2. 高层策略生成子任务指令 $l̂$
3. 世界模型生成子目标图像 $g^* \sim p_ψ(g^* | o_0, l̂, m)$
4. 组合上下文 $C = \{l, l̂, g^*, m, c\}$
5. VLA 模型生成动作块 $a_{t:t+H} \sim π_θ(a | o_{t-T:t}, C)$
6. 每 $∆=4$ 秒或子任务变化时刷新子目标图像
7. 使用分类器自由引导（CFG）增强元数据引导

## 三、实验结果与结论

### 3.1 开箱即用的灵巧操作性能

**核心发现**：π₀.₇ 在多种高难度灵巧任务上，**无需任何任务特定微调**，即可匹配甚至超越专门的 RL 微调专家模型。

| 任务 | π₀.₇ vs RL 专家 |
|------|----------------|
| 衣物折叠（T恤/短裤） | 成功率相当，**吞吐量更高** |
| 衣物折叠（最难物品） | 成功率相当，**吞吐量更高** |
| 制作浓缩咖啡 | 成功率相当 |
| 折叠纸箱 | 成功率相当，**吞吐量更高** |
| 制作花生酱三明治 | 任务进度匹配 |
| 衬衫翻面 | 任务进度匹配 |
| 穿门驾驶 | 任务进度匹配 |
| 切西葫芦 | 任务进度匹配 |
| 削果蔬 | 任务进度匹配 |
| 更换垃圾袋 | 任务进度匹配 |

**消融实验**：移除元数据或移除评估数据后，性能在所有任务上显著下降，证明**元数据标注 + 次优数据蒸馏**是关键。

### 3.2 指令跟随能力

**核心发现**：π₀.₇ 在未见环境中展现出远超 π₀.5 和 π₀.6 的指令跟随能力。

- 4 个未见厨房 + 2 个未见卧室，每个场景 3-6 步指令序列
- π₀.₇ 在指令跟随成功率上**显著优于** π₀.5 和 π₀.6
- 在**复杂指代指令**（如"pick up the fruit on the largest plate"）上表现突出
- **突破数据集偏差**：成功执行"反向清理"等与训练数据相反的指令

### 3.3 跨本体迁移（Cross-Embodiment Transfer）

**核心发现**：π₀.₇ 展现出**零样本跨本体迁移**的涌现能力。

- 源本体：BiPi → 目标本体：UR5e（从未训练过衣物折叠）
- **衣物折叠**等高难度任务成功迁移
- **涌现新策略**：UR5e 采用**单臂拾放**（而非源机器人的双臂操作）和**垂直抓取**（更适合其运动学）

**人类对比实验**：10 名经验丰富的遥操作员首次在 UR5e 上折叠衬衫：
- 人类：成功率 80.6%
- **π₀.₇ (GC)：成功率 80%**
- **性能与人类专家相当！**

### 3.4 组合任务泛化

**核心发现**：π₀.₇ 能通过**语言教练**学习全新任务，无需收集任何新的动作数据。

- **短时任务（零样本）**：按压法压壶活塞、舀米入电饭煲等——**从未出现在机器人训练数据中**
- **长时任务（语言教练）**：通过逐步语言指令执行"Loading an Air Fryer"等全新任务
- **从教练到自主**：收集教练数据后训练高层语言策略，使 π₀.₇ 能**完全自主**执行新任务

### 3.5 数据多样性与缩放规律

**核心发现**：π₀.₇ 能够从**混合质量数据**中持续学习。

- **无元数据**的模型：数据越多、质量越低 → **性能下降**
- **有元数据**的 π₀.₇：数据越多 → **性能持续提升**
- 移除 20% 高任务多样性数据 → 泛化性能**显著下降**
- 随机移除 20% 数据 → 泛化性能保持

## 四、核心结论总结

1. **开箱即用性能**：匹配甚至超越 RL 微调专家，无需任务特定后训练
2. **涌现能力**：跨本体迁移（性能与人类专家相当）、指令理解（突破数据集偏差）、组合泛化（通过语言教练学习新任务）
3. **核心方法**：多样化提示（子任务指令 + 子目标图像 + 片段元数据）使模型能从**大规模混合质量数据**中学习
4. **缩放规律**：元数据标注使模型能从更多（即使质量更低）的数据中受益
5. **实践意义**：灵巧技能可从轻量级低成本平台迁移到工业级机器人，大幅降低数据收集成本
