## ① Q 网络损失 — Soft Bellman Residual

### 先看论文原文

论文公式(2)定义了 Soft Bellman 算子：

$$T^\pi Q(s_t, a_t) \triangleq r(s_t, a_t) + \gamma \mathbb{E}_{s_{t+1}\sim p} [V(s_{t+1})]$$

(Haarnoja 等, 2019)

### 你写的公式

$$J_Q(\theta) = \mathbb{E}_{(s_t,a_t)\sim\mathcal{D}} \left[ \frac{1}{2} \left( Q_\theta(s_t,a_t) - (r(s_t,a_t) + \gamma V_{\bar{\theta}}(s_{t+1})) \right)^2 \right]$$

### 逐层拆解

#### 第1层：什么是 Bellman Residual？

**核心思想：** Q 值必须满足"自洽性"（self-consistency）。

如果 Q 网络是对的，那么：

$$Q(s_t, a_t) \approx r(s_t, a_t) + \gamma \times (\text{下一状态的价值})$$

左边是"当前 Q 值"，右边是"实际观测到的回报"。两者应该相等。

**残差（Residual）** 就是它们的差距：

$$\text{residual} = Q_\theta(s_t,a_t) - \underbrace{(r(s_t,a_t) + \gamma V_{\bar{\theta}}(s_{t+1}))}_{\text{目标值 } y}$$

Q 网络要学习让这个残差接近 0。

#### 第2层：为什么叫 Soft？

普通 Q 学习的 Bellman 方程：

$$Q(s_t,a_t) = r(s_t,a_t) + \gamma \max_{a'} Q(s_{t+1}, a')$$

SAC 的 Soft Bellman 方程：

$$Q(s_t,a_t) = r(s_t,a_t) + \gamma \mathbb{E}_{a'\sim\pi} [Q(s_{t+1}, a') - \alpha \log \pi(a'|s_{t+1})]$$

**区别在哪？**

||普通 Q 学习|SAC（Soft）|
|---|---|---|
|下一状态的价值|$max_{a'} Q(s',a')$ — 选最好的动作|$V(s') = mathbb{E}_{a'simpi}[Q(s',a') - alphalogpi]$ — **期望**，还要减熵|
|含义|确定性最优|**随机性最优**，考虑熵奖励|

#### 第3层：$V(s_{t+1})$ 到底是什么？

你写的第二个公式：

$$V(s_t) = \mathbb{E}_{a_t\sim\pi} \left[ Q(s_t,a_t) - \alpha \log \pi(a_t|s_t) \right]$$

**翻译：** 状态 $s_t$ 的价值 = 所有可能动作的 Q 值的期望，但每个动作要**减去它的熵惩罚**。

**为什么减 $\alpha\log\pi$？**

因为 SAC 的目标不只是最大化奖励，还要最大化熵。所以：

- 一个动作如果 Q 值高 → 好
- 一个动作如果概率高（$\log\pi$ 大，即 $-\alpha\log\pi$ 小）→ 确定性高，熵低 → **惩罚**
- 一个动作如果概率低（$\log\pi$ 小，即 $-\alpha\log\pi$ 大）→ 随机性高，熵高 → **奖励**

所以 $V(s)$ 实际上是 **"带熵奖励的 Q 值期望"**。

#### 第4层：整个 Q 损失在干什么？

```
从经验池采样 (s_t, a_t, r_t, s_{t+1})
         │
         ▼
用目标 Q 网络算 V(s_{t+1}) = E[Q_θ̄(s_{t+1}, a') - α log π(a'|s_{t+1})]
         │
         ▼
构造目标值 y = r_t + γ × V(s_{t+1})
         │
         ▼
算残差 = Q_θ(s_t, a_t) - y
         │
         ▼
损失 = ½ × (残差)²    ← 这就是 MSE（均方误差）
         │
         ▼
梯度下降更新 θ，让 Q_θ(s_t,a_t) 更接近 y
```

**一句话：** Q 网络学习的是"给定 $(s_t,a_t)$，未来能拿到多少带熵奖励的总和"。

## ② Actor 损失 — KL 散度最小化

### 论文原文

论文公式(21)：

$$\pi_{\text{new}}(\cdot|s_t) = \arg\min_{\pi'\in\Pi} D_{\text{KL}}\left( \pi'(\cdot|s_t) \middle\| \frac{\exp(Q^{\pi_{\text{old}}}(s_t, \cdot))}{Z^{\pi_{\text{old}}}(s_t)} \right)$$

(Haarnoja 等, 2019)

### 你写的公式

$$J_\pi(\phi) = \mathbb{E}_{s_t\sim\mathcal{D}} \left[ \mathbb{E}_{a_t\sim\pi_\phi} \left[ \alpha \log \pi_\phi(a_t|s_t) - Q_\theta(s_t,a_t) \right] \right]$$

### 逐层拆解

#### 第1层：这个公式从哪来的？

SAC 的策略更新来自一个**理论保证**：

> 给定旧的 Q 函数 $Q^{\pi_{\text{old}}}$，新的策略 $\pi_{\text{new}}$ 应该让 KL 散度最小化，让 $\pi_{\text{new}}$ 尽可能接近 $\exp(Q^{\pi_{\text{old}}})$ 的形状。

$\exp(Q(s_t, a_t))$ 的意思是：**Q 值越高的动作，概率应该越大**。

#### 第2层：KL 散度怎么变成你写的公式？

KL 散度的定义：

$$D_{\text{KL}}(\pi' \| \exp(Q)) = \mathbb{E}_{a\sim\pi'} [\log \pi'(a|s) - Q(s,a) + \underbrace{\log Z(s)}_{\text{与 }a\text{ 无关}}]$$

去掉与 $a$ 无关的 $\log Z(s)$，再乘以 $\alpha$（温度系数），就得到：

$$J_\pi(\phi) = \mathbb{E}_{s\sim\mathcal{D}, a\sim\pi_\phi} [\alpha \log \pi_\phi(a|s) - Q_\theta(s,a)]$$

#### 第3层：直观理解

这个损失函数要最小化，所以看两项：

|项|想让它|含义|
|---|---|---|
|$alpha log pi_phi(a\|s)$|**小**|$logpi$ 小 → $pi$ 小 → 不要确定地选某个动作？等等，这不对...|
|$- Q_theta(s,a)$|**小**|负的 Q 值小 → Q 值大 → 选 Q 值高的动作|

**等一下，$\alpha\log\pi$ 小意味着 $\pi$ 小，这不是矛盾吗？**

不矛盾。注意这个期望是对 $a\sim\pi_\phi$ 采样的，所以：

- 如果策略 $\pi$ 很确定（$\log\pi \approx 0$），那么 $\alpha\log\pi \approx 0$，损失主要来自 $-Q$
- 如果策略 $\pi$ 很随机（$\log\pi$ 是很大的负数），那么 $\alpha\log\pi$ 是负的，相当于**减了损失**，鼓励随机性

**所以这个损失函数在做两件事的平衡：**

1. 让策略倾向于选 Q 值高的动作（$-Q$ 项）
2. 让策略保持一定的随机性（$\alpha\log\pi$ 项，负值降低损失）

#### 第4层：重参数化技巧

你写的：

$$a_t = f_\phi(\epsilon_t; s_t), \quad \epsilon_t \sim \mathcal{N}$$

**为什么需要这个？**

如果不重参数化，$a_t$ 是从 $\pi_\phi$ 采样的，梯度没法通过"采样"这个随机操作传到 $\phi$。

重参数化把"采样"拆成两步：

```
原来：a ~ π_φ(s)        ← 采样操作不可导
改成：a = μ_φ(s) + σ_φ(s) × ε,  ε ~ N(0,1)  ← 全部可导
```

这样梯度就可以从损失函数一路传到 $\phi$。

#### 第5层：整个 Actor 损失在干什么？

```
对每个状态 s_t：
         │
         ▼
用当前策略采样动作 a = f_φ(ε; s_t)    ← 重参数化
         │
         ▼
算 Q_θ(s_t, a) — 这个动作好不好
算 log π_φ(a|s_t) — 这个动作的确定性
         │
         ▼
损失 = α log π - Q
         │
         ▼
梯度下降更新 φ：
  • 让 Q 大的动作概率更高
  • 让 α log π 保持一定随机性
```

**一句话：** Actor 学习的是"在 Q 值高的地方集中概率，但不要完全确定，留点探索空间"。

## ③ 温度损失 — 自动调节 $\alpha$

### 论文原文

论文从约束优化推导：

$$\max_\pi \mathbb{E}\left[\sum r(s_t,a_t)\right] \quad \text{s.t.} \quad \mathbb{E}[-\log\pi_t(a_t|s_t)] \geq \mathcal{H} \quad \forall t$$

(Haarnoja 等, 2019, 公式11)

通过拉格朗日对偶，得到：

$$\alpha_t^* = \arg\min_\alpha \mathbb{E}_{a_t\sim\pi_t^*} [-\alpha \log\pi_t^*(a_t|s_t) - \alpha\mathcal{H}]$$

(Haarnoja 等, 2019, 公式17)

### 你写的公式

$$J(\alpha) = \mathbb{E}_{a_t\sim\pi_t} \left[ -\alpha \log \pi_t(a_t|s_t) - \alpha \bar{\mathcal{H}} \right]$$

其中 $\bar{\mathcal{H}} = -\dim(\mathcal{A})$ 为目标熵。

### 逐层拆解

#### 第1层：为什么要自动调 $\alpha$？

$\alpha$ 控制"奖励"和"熵"的平衡：

- $\alpha$ 太大 → 策略太随机，不关心奖励
- $\alpha$ 太小 → 策略太确定，不探索

手动调 $\alpha$ 很麻烦，SAC 让它自动学习。

#### 第2层：约束优化怎么变成损失函数？

**原始问题：**

$$\max_\pi \mathbb{E}[\sum r] \quad \text{subject to} \quad \text{熵} \geq \mathcal{H}$$

意思是：在保证熵不低于 $\mathcal{H}$ 的前提下，最大化回报。

**拉格朗日对偶：**

把约束条件变成惩罚项：

$$\max_\pi \min_{\alpha \geq 0} \mathbb{E}[\sum r] + \alpha(\text{熵} - \mathcal{H})$$

展开：

$$\max_\pi \min_{\alpha \geq 0} \mathbb{E}[\sum r] + \alpha \mathbb{E}[-\log\pi] - \alpha\mathcal{H}$$

**对 $\alpha$ 的优化：**

固定 $\pi$，$\alpha$ 要最小化：

$$J(\alpha) = \mathbb{E}[-\alpha\log\pi] - \alpha\mathcal{H}$$

这就是你写的公式。

#### 第3层：直观理解

|情况|当前熵 vs 目标熵|$alpha$ 的梯度|$alpha$ 怎么变|
|---|---|---|---|
|策略太随机|熵 > $mathcal{H}$|$-logpi - mathcal{H}$ 是负的|$alpha$ **减小**，降低熵权重|
|策略太确定|熵 < $mathcal{H}$|$-logpi - mathcal{H}$ 是正的|$alpha$ **增大**，增加熵权重|

**自动调节逻辑：**

- 如果策略太确定（熵不够）→ 增大 $\alpha$，强制它更随机
- 如果策略太随机（熵超标）→ 减小 $\alpha$，让它更关注奖励

#### 第4层：目标熵 $\bar{\mathcal{H}} = -\dim(\mathcal{A})$ 为什么是这个值？

论文附录里提到，目标熵通常设为 $-\dim(\mathcal{A})$。

**直觉：** 如果动作空间有 $d$ 维，均匀分布在一个 $[-1,1]^d$ 的立方体上的熵大约是 $d$（具体取决于分布）。$-\dim(\mathcal{A})$ 是一个经验值，对应一个**适度随机**的策略。

```
└────────────────────┬────────────────────────────────────┘
                     │
                     ▼
        ┌────────────────────────────┐
        │    采样一个 batch          │
        └────────┬───────────────────┘
                 │
    ┌────────────┼────────────────────┐
    │            │                    │
    ▼            ▼                    ▼
┌─────────┐ ┌─────────┐        ┌──────────┐
│ Q 损失   │ │ Actor   │        │ 温度损失  │
│         │ │ 损失    │        │          │
│ 让 Q_θ  │ │ 让 π_φ  │        │ 让 α     │
│ 逼近    │ │ 靠近    │        │ 自动调节  │
│ r+γV    │ │ exp(Q)  │        │ 熵达标   │
└─────────┘ └─────────┘        └──────────┘
     │            │                  │
     ▼            ▼                  ▼
 更新 θ       更新 φ             更新 α
（Q网络）    （策略网络）        （温度）
     │            │                  │
     └────────────┼──────────────────┘
                  ▼
           EMA 软更新 θ̄
          （目标 Q 网络）
```

**三个损失的分工：**

|损失|更新谁|目标|
|---|---|---|
|$J_Q(theta)$|Q 网络 $theta$|学会准确估计"带熵的未来回报"|
|$J_pi(phi)$|策略网络 $phi$|在 Q 值高的地方集中概率，同时保持随机|
|$J(alpha)$|温度 $alpha$|自动调节"奖励 vs 熵"的平衡|

**训练循环：**

1. Q 损失让 Q 网络学会**评价**动作好坏
2. Actor 损失让策略网络学会**生成**好动作
3. 温度损失自动调节**探索程度**
