# PPGFlowECG: Latent Rectified Flow with Cross-Modal Encoding for PPG-Guided ECG Generation and Cardiovascular Disease Detection

## 潜在空间（Latent Space）到底是什么？

### 核心直觉：一个"语义翻译器"

PPG 和 ECG 是两种不同形态的信号，但反映的是同一套心血管动力学。潜在空间就是把这两种信号"翻译"成一种共享的、语义对齐的中间表示。

> PPG and ECG differ in surface morphology yet reflect the same underlying cardiovascular dynamics.

(Fang et al., 2026)

**类比**：A 说中文（PPG），B 说英文（ECG），他们在讨论同一个话题（心脏的生理活动）。潜在空间就是"中间语言"——把中文和英文都翻译成语义向量，在这个中间语言里，"心跳快"和"tachycardia"被映射到同一个位置。

### 数学定义

潜在空间是一个**低维向量空间**：

1. **输入**：PPG 信号 $x_{ppg} \in \mathbb{R}^{L \times 1}$（1280 维时间序列）
2. **编码器**：$E_{CA}(\cdot)$ 把 1280 维波形压缩成低维向量
3. **输出**：均值 $\mu_{ppg}$ 和标准差 $\sigma_{ppg}$，通过重参数化采样得到潜在变量 $z_{ppg}$

$$z_{ppg} = \mu_{ppg} + \sigma_{ppg} \odot \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, I)$$

这个 $z_{ppg}$ 就是 PPG 信号在**潜在空间中的坐标**。同样，ECG 信号也被映射到同一个空间中的坐标 $z_{ecg}$。

### 为什么叫"潜在"（Latent）？

"潜在"的意思是**隐藏在表面之下的东西**：

- **表面**：PPG 的波形（起伏的曲线）或 ECG 的波形（P波、QRS波群、T波）
- **潜在**：波形背后隐藏的**生理状态**——心率、心律、心肌电活动、血管状态……

核心思想：PPG 和 ECG 的表面波形不同，但它们背后的"潜在生理状态"是相同的。潜在空间就是捕获这个"隐藏的生理状态"。

### 三层次对齐：如何构建潜在空间

#### ① 全局分布对齐（粗粒度）

把 PPG 和 ECG 的潜在表示看作两个高斯分布，让它们的均值靠近、方差匹配：

$$L_{GDA} = \|\mu_{ecg} - \mu_{ppg}\|_2^2 + \frac{1}{2}\left[D_{KL}(q_{ecg} \| q_{ppg}) + D_{KL}(q_{ppg} \| q_{ecg})\right]$$

**直观理解**：让中文和英文的"整体语义分布"大致对齐。

#### ② 局部实例判别（细粒度）

用对比学习（InfoNCE 损失），让同一受试者的 PPG 和 ECG 潜在表示靠近，不同受试者的远离：

$$L_{LID} = -\frac{1}{2}\left[\log\frac{e^{\langle \bar{z}_{ecg}, \bar{z}_{ppg} \rangle / \tau}}{\sum_j e^{\langle \bar{z}_{ecg}, \bar{z}_{ppg}^{(j)} \rangle / \tau}} + \log\frac{e^{\langle \bar{z}_{ppg}, \bar{z}_{ecg} \rangle / \tau}}{\sum_j e^{\langle \bar{z}_{ppg}, \bar{z}_{ecg}^{(j)} \rangle / \tau}}\right]$$

**直观理解**：在词典对齐的基础上，确保"张三说的'心跳快'"和"张三写的'tachycardia'"对应同一个向量。

#### ③ 语义可解码性约束（最强约束）

要求 $z_{ppg}$ 能解码出 ECG 波形，$z_{ecg}$ 能解码出 PPG 波形：

$$L_{SDC} = \|D_{ECG}(z_{ppg}) - x_{ecg}\|_2^2 + \|D_{PPG}(z_{ecg}) - x_{ppg}\|_2^2$$

**直观理解**："你说你理解了中文的意思，那请你用英文复述一遍"——如果 $z_{ppg}$ 真的捕获了生理语义，它应该能重建出 ECG 波形。

### 为什么在潜在空间做生成，而不是直接在原始波形？

论文给出了形式化的数学证明。

定义**条件分散度**（condition dispersion）：

$$\bar{\kappa} = \mathbb{E}_c\left[\text{tr}\left(\text{Cov}(y | c)\right)\right]$$

它衡量的是：给定 PPG 潜在 $c$ 时，ECG 潜在 $y$ 的不确定性有多大。

Stage 1 的三层次对齐使 $\bar{\kappa} \to 0$（PPG 和 ECG 在潜在空间中几乎重合）。

Stage 2 流匹配的**不可约误差下界**为：

$$R_{\min} \leq \bar{\kappa}$$

**翻译成人话**：Stage 1 把潜在空间对齐得越好，Stage 2 生成 ECG 的误差上限就越低。如果直接在原始波形空间做生成，$\bar{\kappa}$ 很大（PPG 和 ECG 波形差异巨大），生成质量就会差。

### 整体架构图

```
原始波形空间（高维、非平滑）：
  PPG 波形 ──────────────────────── ECG 波形
  (形态完全不同)                    (形态完全不同)
        │                              │
        │  CardioAlign Encoder         │
        ▼                              ▼
潜在空间（低维、语义对齐）：
  z_ppg ──────── 几乎重合 ──────── z_ecg
        │                              │
        │  流匹配（直线运输）           │
        │  z → y                       │
        ▼                              ▼
  生成的 ECG 潜在 ──→ ECG 解码器 ──→ ECG 波形
```

**潜在空间就是那个"中间翻译层"**——它把形态完全不同的 PPG 和 ECG 统一到同一个语义坐标系中，让系统不再关心"波形长什么样"，而是关心"心脏在做什么"。
