# TFCDiff: Robust ECG Denoising via Time-Frequency Complementary Diffusion

## 扩散模型 vs 流匹配（Flow Matching）

### 一、论文中提到的流匹配

在论文的 **Limitations and Future Works** 部分，作者写道：

> "We observe unsatisfactory denoising results for signals with intensive EMG noise. This is likely attributable to the similar distribution between EMG noise and ECG signals. **Efforts could be directed toward exploring flow-matching techniques [27], which offer greater flexibility for modeling arbitrary distribution.**"

(Li et al., 2025)

也就是说，作者认为流匹配的**优势在于建模任意分布时更灵活**，而扩散模型在处理与 ECG 分布相似的 EMG 噪声时效果不佳。

---

### 二、核心区别

#### 2.1 数学框架的本质差异

| 方面 | 扩散模型 | 流匹配 |
|:---|:---|:---|
| **路径类型** | **随机过程**（Stochastic）—— 每一步都加随机噪声 | **确定性过程**（Deterministic）—— 连续平滑的确定性变换 |
| **理论基础** | 非平衡热力学 + 变分下界（ELBO） | 连续归一化流（CNF）+ 最优传输 |
| **训练目标** | 预测噪声 $\varepsilon_\theta(x_t, t, \tilde{x})$ | 预测速度场 $v_\theta(x_t, t)$ |
| **采样方式** | 迭代去噪（通常 50~1000 步） | ODE 求解（可少至 10 步） |
| **噪声处理** | 显式建模噪声分布 $\mathcal{N}(0, I)$ | 隐式学习从噪声到数据的映射 |

#### 2.2 路径对比

**扩散模型（随机路径）：**

```
x₀ ──加噪──→ x₁ ──加噪──→ x₂ ──→ ... ──→ x_T (纯噪声)
  ←──去噪──   ←──去噪──   ←──      ←──
    每一步都有随机性 σ_t·z
```

路径是**随机游走**，每一步 $x_t$ 都是从高斯分布中采样得到的，不是唯一确定的。

**流匹配（确定性路径）：**

```
x₀ ──────────────────────────────────────→ x₁ (纯噪声)
  ←──────────────────────────────────────
    一条平滑的确定性轨迹，没有随机性
```

路径是**一条平滑的曲线**，由速度场 $v_\theta(x_t, t)$ 唯一确定，没有随机抖动。

#### 2.3 数学公式对比

##### 扩散模型

**前向过程（离散时间步）：**
$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, I)$$

**反向过程（去噪）：**
$$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{1 - \alpha_t}{\sqrt{1 - \bar{\alpha}_t}} \varepsilon_\theta(x_t, t, \tilde{x}) \right) + \sigma_t z$$

**训练损失：**
$$\mathcal{L} = \mathbb{E}[\| \varepsilon - \varepsilon_\theta(x_t, t, \tilde{x}) \|_1]$$

##### 流匹配

**前向过程（连续时间，ODE 形式）：**
$$\frac{dx_t}{dt} = v(x_t, t)$$

其中 $v$ 是速度场（velocity field），$t \in [0, 1]$。

**从数据到噪声的线性插值路径：**
$$x_t = (1 - t) \cdot x_0 + t \cdot \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, I)$$

**训练损失（预测速度场）：**
$$\mathcal{L} = \mathbb{E}[\| (x_1 - x_0) - v_\theta(x_t, t) \|^2]$$

其中 $x_1 - x_0 = \varepsilon - x_0$ 是"从数据到噪声的方向向量"。

**采样（反向 ODE 求解）：**
$$\frac{dx_t}{dt} = v_\theta(x_t, t), \quad \text{从 } x_1 \text{ 积分到 } x_0$$

### 三、关键差异总结

#### 3.1 训练目标不同

| | 扩散模型 | 流匹配 |
|:---|:---|:---|
| **学什么** | 学**噪声** $\varepsilon_\theta$ | 学**速度场** $v_\theta$ |
| **损失函数** | $\|\varepsilon - \varepsilon_\theta\|_1$ | $\|(x_1 - x_0) - v_\theta\|^2$ |
| **物理意义** | 猜"当前信号里混入了多少噪声" | 猜"从数据到噪声的方向和速度" |

#### 3.2 采样过程不同

| | 扩散模型 | 流匹配 |
|:---|:---|:---|
| **采样方式** | 离散迭代去噪（50 步） | ODE 连续积分（可少至 10 步） |
| **随机性** | 每一步加随机抖动 $\sigma_t z$ | 无随机性，纯确定性 |
| **速度** | 较慢（50 步 × 网络推理） | 较快（可用更少步数） |

#### 3.3 分布建模能力

| | 扩散模型 | 流匹配 |
|:---|:---|:---|
| **假设** | 假设噪声是高斯分布 $\mathcal{N}(0, I)$ | 无分布假设 |
| **灵活性** | 受限于高斯噪声假设 | **更灵活**，可建模任意分布 |
| **适用场景** | 噪声接近高斯分布时效果好 | 噪声分布复杂/未知时更有优势 |

---

### 四、为什么论文建议未来探索流匹配？

论文指出，TFCDiff 在处理**密集 EMG 噪声**时效果不理想，原因是：

> EMG 噪声的分布与 ECG 信号的分布**高度相似**，扩散模型的高斯噪声假设 $\varepsilon \sim \mathcal{N}(0, I)$ 无法很好地建模这种复杂分布。

而流匹配的优势在于：

1. **不假设噪声是高斯分布**—— 速度场 $v_\theta$ 可以学习任意复杂的分布变换
2. **路径更灵活**—— 线性插值路径 $(1-t)x_0 + t\varepsilon$ 只是最简单的选择，还可以用最优传输路径
3. **采样更快**—— ODE 求解器可以用更少的步数达到相同质量

#### 直观类比

| | 扩散模型 | 流匹配 |
|:---|:---|:---|
| **类比** | 像**随机漫步**下山，每一步都随机晃一下 | 像**滑滑梯**下山，沿着一条平滑的轨道 |
| **路径** | 锯齿状、随机 | 平滑、确定 |
| **步数** | 需要很多步（50） | 可以用更少步（10-20） |
| **灵活性** | 轨道固定（高斯噪声假设） | 轨道可定制（任意分布） |

---

### 五、一句话总结

> **扩散模型** = 离散随机过程，学噪声，高斯假设，50 步去噪
> **流匹配** = 连续确定性过程，学速度场，无分布假设，ODE 积分，更快更灵活

论文认为流匹配是未来改进方向，因为它在处理**与 ECG 分布相似的 EMG 噪声**时，比扩散模型的高斯假设更具灵活性。
