## 2. 理想二极管 (The Ideal Diode)
![[BAO_Ch4_diodes20260302(3).pdf#page=4]]
![[BAO_Ch4_diodes20260302(3).pdf#page=5]]
![[BAO_Ch4_diodes20260302(3).pdf#page=6]]

* **什么是理想二极管？** 它是一个最基础的非线性电路元件，只有两个端子：阳极（Anode，流入端）和阴极（Cathode，流出端）。
* **两种工作模式**：
    * **反向偏置 (Reverse bias)**：当电压 v < 0 时，电流 i = 0。此时二极管就像一个**断开的开关 (open circuit)**。
    * **正向偏置 (Forward bias)**：当电流 i > 0 时，电压 v = 0。此时二极管就像一个**闭合的开关 (short circuit)**。
* **注意事项**：在实际电路中，必须外接电阻来限制正向导通时的电流和反向截止时的电压，否则二极管会烧毁。

![[BAO_Ch4_diodes20260302(3).pdf#page=7]]
![[BAO_Ch4_diodes20260302(3).pdf#page=8]]
![[BAO_Ch4_diodes20260302(3).pdf#page=9]]
* **简单应用：整流器 (Rectifier)**。整流器的作用是把交流电（AC）无损地变成直流电（DC）。因为它只允许电流单向通过，所以可以把正弦波的负半周给“砍掉”。

* **例题 4.1 (Example 4.1)**：用峰值 24V 的交流源通过二极管给 12V 电池充电。
    * **通俗解析**：只有当交流电压源 v_S 的电压高于电池电压 12V 时，二极管才导通。此时正向电流 i_D = (v_S - 12) / 100欧姆。最大反向电压发生在交流电负半波峰值（-24V）时，此时二极管承受的电压是 -24V - 12V = -36V。

## 3. 实际结型二极管的特性 (Terminal Characteristics of Junction Diodes)
![[BAO_Ch4_diodes20260302(3).pdf#page=10]]
![[BAO_Ch4_diodes20260302(3).pdf#page=11]]
![[BAO_Ch4_diodes20260302(3).pdf#page=12]]
![[BAO_Ch4_diodes20260302(3).pdf#page=13]]
![[BAO_Ch4_diodes20260302(3).pdf#page=14]]
![[BAO_Ch4_diodes20260302(3).pdf#page=15]]

实际由 PN 结构成的二极管，它的 I-V 曲线分为三个区域：

1.  **正向偏置区 (Forward-Bias Region)**：当 v > 0 时。电流和电压呈指数关系：i = I_S(e^(v/V_T) - 1)。
    * I_S 是饱和电流，由温度决定。
    * V_T 是热电压，在室温下约为 25.8mV。
    * **开启电压 (Cut-in voltage)**：大约在 0.5V 以下时，电流非常小。当电压达到 0.6V ~ 0.8V 时，二极管充分导通，电阻接近于 0。
2.  **反向偏置区 (Reverse-Bias Region)**：当 v < 0 时。指数项变小，电流 i 约等于 -I_S。因为 I_S 极小（通常是 10^-15 A 级别），所以近似认为断路。
3.  **击穿区 (Breakdown Region)**：当反向电压超过一个阈值 V_ZK 时，电流会剧增。如果不烧毁，这个特性常用于稳压（齐纳二极管）。

## 4. 二极管建模方法 (Modeling the Diode Characteristic)
在电路分析中，每次都解指数方程太痛苦了，所以我们有三种不同的模型来简化计算。

### 4.1 指数模型与图解法 (Exponential Model & Graphical Analysis)

![[BAO_Ch4_diodes20260302(3).pdf#page=17]]
![[BAO_Ch4_diodes20260302(3).pdf#page=18]]
![[BAO_Ch4_diodes20260302(3).pdf#page=19]]
![[BAO_Ch4_diodes20260302(3).pdf#page=20]]
* 最精确，但也最难算，因为是非线性方程。
* **图解法 (Graphical Analysis)**：画出二极管的指数曲线，再画出外围电路的“负载线” (Load line) I_D = (V_DD - V_D) / R。两条线的交点 Q 就是电路的工作点 (Operating point)。
* 优点是直观，缺点是精度差且不适合复杂电路。

### 4.2 恒压降模型 (Constant Voltage-Drop Model, CVDM)
![[BAO_Ch4_diodes20260302(3).pdf#page=21]]
![[BAO_Ch4_diodes20260302(3).pdf#page=22]]
* **最常用的工程模型**：直接假设只要二极管导通，两端的压降就恒定为 0.7V。
* 这把复杂的指数曲线变成了一条在 0.7V 处垂直向上的直线。极大地简化了计算，兼顾了实用性和一定的准确性。

### 4.3 理想二极管模型 (Ideal Diode Model)
![[BAO_Ch4_diodes20260302(3).pdf#page=23]]
![[BAO_Ch4_diodes20260302(3).pdf#page=24]]
![[BAO_Ch4_diodes20260302(3).pdf#page=25]]
* 假设导通时压降为 0V。
* **何时使用？** 当电路中的外部电压远大于 0.7V（比如几百伏）时，这 0.7V 的误差就可以忽略不计了。

当二极管上既有直流电源，又叠加了一个微小的交流信号（如音频信号）时，怎么算？
* **核心思想（线性化）**：把瞬时总信号拆分为“静态直流 (DC)”和“动态交流 (Time-varying)”独立分析。
* **第一步：算直流工作点**。只看直流电源，把二极管当成 0.7V 的恒压降模型 (CVDM)，算出静态电流 I_D。
* **第二步：算小信号电阻**。利用泰勒展开（由于交流电压 v_d 远小于 V_T ，高次项可忽略）。在工作点附近，二极管等效为一个动态电阻 r_d：
    r_d = V_T / I_D
    （通常 V_T 取 25mV）。
* **第三步：画交流等效电路**。把直流电源短路接地，把二极管替换为电阻 r_d，用分压定理直接计算交流信号。

**例题 4.5 解析**：10V 直流叠加 1V 交流纹波，通过 10k 欧姆电阻。
1. 直流分析：I_D = (10V - 0.7V) / 10k = 0.93mA。
2. 交流电阻：r_d = 25mV / 0.93mA = 26.9欧姆。
3. 交流分压：二极管上的交流峰值 v_d = 1V * (26.9 / (10000 + 26.9)) 约等于 2.68mV。

![[BAO_Ch4_diodes20260302(3).pdf#page=38]]

* 什么是稳压器？即使电源波动或负载改变，输出电压依然保持稳定的电路。
* 利用二极管的正向压降几乎恒定为 0.7V 的特性，将多个二极管串联（如3个串联得到 2.1V）就可以做一个简单的稳压器。

### 4.1 电池接理想二极管
* **情况 (a) 阴极接正极**：二极管反偏，相当于断路。电流 I = 0，二极管两端电压（从阳极到阴极）为 -1.5V。
* **情况 (b) 阳极接正极**：二极管正偏，理想情况下压降为 0V。此时电流由电池内阻决定：I = 1.5V / 1欧姆 = 1.5A。

### 4.2 理想二极管电路状态判断
*(假设二极管为理想状态，正偏 V=0，反偏 I=0)*
* **(a)**：阳极 0V，阴极通过电阻接 -5V。有导通趋势，二极管正偏。V = 0V，I = (0 - (-5)) / 10k = 0.5mA。
* **(b)**：阳极 -5V，阴极 0V。二极管反偏断路。I = 0A。由于没有电流流过电阻，V点的电压等于阳极电压，所以 V = -5V。
* **(c)**：阳极 +5V，阴极通过电阻接 0V。正偏。节点 V 与地相连，所以 V = +5V（阳极直接接5V，二极管短路，V就是5V）。I = (5 - 0) / 10k = 0.5mA。
* **(d)**：阳极通过电阻接 0V，阴极接 +5V。反偏断路。I = 0A。电阻无压降，V点等于 +5V。

### 4.3 多个理想二极管竞争导通
* **(a)**：D1和D2的阴极连在一起，接在同一个节点 V。D2的阳极电压更高（+2V），所以D2会优先导通，将节点 V 钳位在 +2V。此时D1的阳极为+1V，阴极为+2V，D1被反偏截止。因此 V = 2V，电流仅流过D2：I = (2V - (-5V)) / 2k = 3.5mA。
* **(b)**：D1和D2的阳极连在一起，接在同一个节点 V。D1的阴极电压更低（+1V），优先导通，将节点 V 钳位在 +1V。此时D2阳极+1V，阴极+2V，反偏截止。因此 V = 1V，电流 I = (+5V - 1V) / 2k = 2mA。

### 4.10 戴维南等效与二极管
* **(a)**：图左侧是一个 +6V 的分压器（两个10k），从中间看进去，**左侧戴维南等效电压**为 6V * (10k/20k) = 3V，内阻为 10k || 10k = 5k欧姆。**右侧戴维南等效电压**为 5V * (10k/20k) = 2.5V，内阻为 5k欧姆。
    * 因为左侧(3V) > 右侧(2.5V)，所以中间的二极管（阳极在左）正向导通。
    * 假设是理想二极管，导通压降为0。等效电路的电流为：I_diode = (3V - 2.5V) / (5k + 5k) = 0.5V / 10k = 0.05mA。
    * 注意图中电流 I 的箭头向左，所以 I = -0.05mA。节点 V 的电压为右侧等效电路电压加上内阻压降：V = 2.5V + 0.05mA * 5k = 2.75V。

### 4.13 半波整流器分析
输入是对称方波，峰峰值4V（即 ±2V），R = 100欧姆。
* **峰值输出电压**：正半周+2V时二极管导通（假设为理想二极管），所以输出峰值为 +2V。
* **平均输出电压**：半个周期是 2V，半个周期是 0V。平均值 = 1V。
* **峰值二极管电流**：2V / 100欧姆 = 20mA。
* **平均二极管电流**：只有半个周期有电流，平均值 = 10mA。
* **最大反向电压**：负半周输入为 -2V，此时二极管截止，承受全压 2V。

### 4.21 估算饱和电流 I_S 与电压变化
利用公式 V_2 - V_1 = V_T * ln(I_2 / I_1)。当电流变为原来的 10% (即下降一个数量级， decade) 时，电压的变化量为 ΔV = V_T * ln(0.1) 约等于 2.3 * 25mV = 57.5mV，近似为 60mV。因此，电流降为 10%，电压下降 60mV。
由 I = I_S * e^(V/V_T) 可得 I_S = I * e^(-V/25mV)：
* **(a)** I_S = 1 * e^(-700/25) = 6.9 * 10^-13 A。电压 = 0.7V - 0.06V = 0.64V。
* **(b)** I_S = 10^-3 * e^(-650/25) = 5.1 * 10^-15 A。电压 = 0.65V - 0.06V = 0.59V。

### 4.46 小信号交流衰减器证明
大电容 C_1, C_2 阻断了直流，但对交流短路。
直流源 I 为二极管提供了静态偏置点，使得其动态电阻为 r_d = V_T / I。
画出交流等效电路时，直流源 I 视为开路（无穷大电阻），只有 R_s 和 r_d 串联分压：
v_o = v_s * (r_d / (R_s + r_d)) = v_s * ((V_T/I) / (R_s + V_T/I)) = v_s * (V_T / (V_T + I * R_s))
得证。若令 v_o = 0.5 * v_s，意味着分压比例为一半，即 r_d = R_s => V_T / I = R_s => I = V_T / R_s = 25mV / 1k = 25μA。
